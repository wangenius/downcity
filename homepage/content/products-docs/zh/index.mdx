# 产品文档

Products Docs 只讲 AI builders 会直接操作的产品表层。它们都建立在 Downcity 共享 runtime 之上：本地 Agent 宿主、浏览器控制面、网页上下文采集、SDK 和可复用 UI。

## 产品入口

- [Town CLI](/zh/products-docs/town-cli/init)：安装、初始化、启动、停止、配置和运维本机 Agent。
- [Town Console](/zh/products-docs/town-console)：在浏览器控制台里管理 agents、models、plugins 和运行状态。
- [Chrome Extension](/zh/products-docs/chrome-extension/overview)：把网页连接到 Town，并把页面上下文送给 agent。

## 什么时候切到其他 Space

- 构建或部署共享服务 runtime，读 [共享服务 SDK](/zh/city-sdk-docs)。
- 嵌入可编程 Agent，读 [Agent SDK](/zh/agent-sdk-docs)。
- 添加账号、余额、usage 或支付能力，读 [Services SDK](/zh/services-sdk-docs)。
