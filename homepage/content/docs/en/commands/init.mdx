# `bay init`

Initialize global console infrastructure.

## Usage

```bash
bay init
```

## Behavior

- Writes/updates `~/.downcity/downcity.db` (global console infrastructure)
- Generates `~/.downcity/schema/downcity.schema.json` (editor schema)

## Notes

- This is a console-level command and does not depend on an agent project path.
- For project-level configuration, use [`bay config`](/en/docs/commands/config).
- `bay init` does not create or update models.
- Configure models later with `bay model create`.
