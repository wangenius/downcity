# `sma status`

Check SMA manager status and active managed agent daemons.

## Usage

```bash
sma status
```

## Output behavior

- Shows whether manager process is running
- Shows manager `pidFile` and `log` paths
- Shows active managed agent daemons summary

## Examples

```bash
# manager global status
sma status

# project daemon status (use agent subcommand)
sma agent status /path/to/agent
```
