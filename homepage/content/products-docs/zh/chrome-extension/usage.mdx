# 使用说明

## Side Panel 怎么用

1. 点击浏览器工具栏中的扩展图标
2. 在 Popup 中点击 `侧栏`
3. 选择目标 Agent
4. 输入消息并发送

Side Panel 会为当前 Town 和 Agent 自动维护一个稳定的浏览器会话，所以你可以边浏览边持续对话。

如果勾选 `附带当前页面`，本次消息会同时带上当前网页的 Markdown 快照。

## Extension Popup 怎么用

1. 点击浏览器工具栏中的扩展图标
2. 选择目标 Agent
3. 输入任务说明
4. 点击发送，或按 `Cmd/Ctrl + Enter`

Popup 适合做明确、完整的一次网页投递。

## 发送时实际发生了什么

扩展会按下面的顺序工作：

1. 读取当前网页标题和 URL
2. 抓取页面正文并转换成 Markdown
3. 生成一个附件文件
4. 发送给选定 Agent

## 页面历史有什么用

Popup 会记录当前页面在当前 Town 下最近的发送历史，方便你：

- 回看之前发过什么
- 避免重复输入
- 确认自己从这个页面发过什么

## 常用快捷键

- `Cmd/Ctrl + Enter`：发送当前 Ask
