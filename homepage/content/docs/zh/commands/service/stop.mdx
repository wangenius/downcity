# `sma service stop`

停止目标 agent 下的指定 service。

## 用法

```bash
sma service stop <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
sma service stop task --agent my-agent
```
