# 通过 HTTP 暴露 Agent

如果你的调用方和执行方不在同一个进程里，可以把本地 Agent 暴露成 HTTP 服务。

在 Downcity 里，HTTP 暴露由 Downcity 发布：

```bash
downcity agent start
```

然后别的进程用：

```ts
new RemoteAgent({
  url: "http://127.0.0.1:5314/agents/repo-helper",
})
```

去连接它。

这是一种很自然的本地 client-server 形态：

- 本地 Agent 负责执行
- Downcity 负责 HTTP 网关发布
- RemoteAgent 负责消费

如果你同时注册了 plugins，Downcity 也会把 plugin 的 HTTP 路由一起挂出去。
