# Restart an Agent Safely

If you changed:

- `downcity.json`
- model binding
- channel binding
- environment variables

the recommended sequence is:

```bash
bay agent status
bay agent restart
bay agent status
```

This is easier to observe than restarting blindly.

Especially when you changed models, channels, and prompts at the same time, checking state before and after restart is a much safer routine.
