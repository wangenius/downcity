# Your First Agent Chat

After the agent starts, run:

```bash
bay agent chat
```

If multiple agents are running in the current environment, the CLI lets you choose one. If you want a repeatable workflow, use `--to <id>` explicitly.

If you want the deeper CLI chat, channel, and SDK material, continue with:

- [CLI chat](/en/docs/agent/cli/interaction/chat)
- [Connect chat channels](/en/docs/agent/getting-started/connect-chat-channels)
- [Agent SDK](/en/agent-sdk-docs)
