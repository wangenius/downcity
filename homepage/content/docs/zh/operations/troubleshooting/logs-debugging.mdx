# 日志与调试

## 1) 看运行日志

```bash
tail -f .downcity/logs/$(date +%F).jsonl
```

## 2) 看 plugin 状态

```bash
bay plugin list
```

## 3) 看会话数据

```bash
bay chat context --chat-key ctx_xxxxxxxxxxxxxxxx
bay chat history --chat-key ctx_xxxxxxxxxxxxxxxx --limit 50
```

## 4) 看任务执行产物

```bash
ls -la .downcity/task/<taskId>
ls -la .downcity/task/<taskId>/<timestamp>
```

`run.json` + `result.md` 基本可以还原一次任务执行结果。
