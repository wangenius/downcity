# `bay agent doctor`

Diagnose agent state files for a target agent and optionally clean stale pid/meta files.

> This page remains as a command entry point. For the fuller explanation, see [CLI / doctor](/en/docs/agent/cli/maintenance/doctor) and [Agent Shows stale](/en/docs/agent/operations/troubleshooting/agent-shows-stale).

## Usage

```bash
bay agent doctor [path] [--fix]
```

## Key option

- `--fix [enabled]`: clean stale state files automatically

## Behavior

- `path` defaults to current directory `.`
- Target agent must be registered in console registry
- Without `--fix`, it only reports issues and does not modify files

- [`bay agent status`](/en/docs/commands/agent/status)
