# 快速开始

## 1. 安装

```bash
npm install -g shipmyagent
shipmyagent --version
```

> 可用短命令 `sma`，与 `shipmyagent` 等价。

## 2. 初始化项目

```bash
cd /path/to/your-repo
sma init
```

初始化后会生成：

- `PROFILE.md`（必需）
- `SOUL.md` / `USER.md`（可选）
- `ship.json`
- `.ship/` 运行目录

## 3. 配置 LLM

在项目根目录创建 `.env`（不要提交到 Git）：

```bash
LLM_API_KEY=your_key
```

`ship.json` 最小示例：

```json
{
  "name": "my-agent",
  "version": "1.0.0",
  "llm": {
    "activeModel": "default",
    "providers": {
      "default": {
        "type": "anthropic",
        "apiKey": "${LLM_API_KEY}"
      }
    },
    "models": {
      "default": {
        "provider": "default",
        "name": "claude-sonnet-4-5"
      }
    }
  }
}
```

## 4. 启动运行时

启动（默认后台 daemon，适合长期运行）：

```bash
sma agent on
```

需要在当前终端前台观察日志时再用：

```bash
sma agent on --foreground
```

## 5. 验证服务状态

```bash
curl http://localhost:3000/health
curl http://localhost:3000/api/status
```

## 6. 可选：配置命令别名

```bash
sma config alias
```
