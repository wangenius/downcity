Contact service

关键点：
- contact 是 agent 之间的点对点关系，不是群组。
- `contact link` 生成一次性 link code，由人转交给另一个 agent。
- `contact approve <code>` 使用 link code 建立双向可信 contact。
- 每个 contact 固定一条长期 chat history，不需要维护额外 session。
- `contact share` 用于分享文本、链接、文件和目录。
- 收到的 share 只进入 inbox，必须显式 `contact receive` 后才会进入 received 区。
