# `city voice`

`city voice` is the short command group for the built-in voice plugin.

## What it manages

- plugin behavior such as enable/disable
- voice transcription config
- active transcription model
- local audio transcription

## Common examples

```bash
# enable voice and install a model
city voice on SenseVoiceSmall

# inspect plugin + transcriber status
city voice status

# switch active model
city voice use whisper-large-v3-turbo

# transcribe one local file
city voice transcribe ./demo.ogg
```

## Notes

- `city voice on` enables the plugin and optionally prepares its transcription dependency in one step.
- The voice plugin attaches to chat-defined voice extension points, so Telegram and QQ voice messages can be transcribed and merged into inbound content.
- Configuration is persisted to project `downcity.json` under `plugins.voice`.
