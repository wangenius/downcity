# 内置环境变量

CLI 在运行 agent 相关命令时，会推导当前 agent 上下文，并注入一些环境变量。

最常见的是：

- `DC_AGENT_PATH`
- `DC_AGENT_NAME`

## `DC_AGENT_PATH`

当前 agent 项目的绝对路径。

它在脚本、日志和一些调试场景里都很有价值。

## `DC_AGENT_NAME`

当前 agent 的显示名称。

通常优先来自：

- `downcity.json.name`

如果没有，再回退到目录名。

这些变量是命令执行时按当前上下文注入的，不等于它们会永久回写到你当前 shell 里。
