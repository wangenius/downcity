# `session.system()`

## 示例

```ts
const system = await session.system();
```

## 它返回什么

返回当前 session 生效的 system prompt 文本数组，类型是 `string[]`。

这个数组包括：

- 项目静态 system prompt
- 显式注入 service 的 system 文本
- 已注册 plugin 的 system 文本
- runtime clock prompt

返回内容和 `session.run()`、`session.stream()` 使用的是同一套 SDK system composer。

## 它不会做什么

`session.system()` 不会把 system prompt 写进 `messages.jsonl`。

Session history 仍然只关注可见的 user / assistant 消息。如果你需要检查当前会随历史一起进入模型的提示词上下文，用 `system()`。
