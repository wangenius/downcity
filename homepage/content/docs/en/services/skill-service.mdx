# Skill Service

`skill` service manages local learned skills.

## Responsibilities

- Find skills
- Install skills
- List learned skills
- Lookup `SKILL.md` content for execution context injection

## Runtime operations

```bash
sma service status skill
sma service command skill list --payload '{}'
```
