# `sma voice`

管理本地语音识别扩展 `voice`。

## 用法

```bash
sma voice <subcommand> [options]
```

## 子命令（当前版本）

- `models`
- `status`
- `on [models...]`
- `init [models...]`
- `off`
- `install [models...]`
- `use <modelId>`
- `transcribe <audioPath>`

## `on` 常用参数

- `--models-dir <path>`：模型目录（默认 `~/.ship/models/voice`）
- `--active-model <modelId>`：激活模型
- `--no-install`：只写配置不下载
- `--force`：覆盖已有模型文件
- `--hf-token <token>`：私有/Gated 模型下载 token

## `init` 常用参数

- `[models...]`：模型列表（不传则在交互终端弹出多选）
- `--active-model <modelId>`：激活模型（不传且多模型时会交互选择）
- `--no-install-model`：跳过模型下载，仅写配置
- `--no-install-deps`：跳过 Python 转写依赖安装
- `--python <bin>`：指定 Python 可执行文件
- `--venv-dir <path>`：PEP 668 回退虚拟环境目录
- `--no-pip-upgrade`：安装依赖时不带 `pip -U`
- `--pip-timeout-ms <ms>`：pip 安装超时

## `install` 常用参数

- `--models-dir <path>`
- `--force`
- `--hf-token <token>`

## `transcribe` 常用参数

- `<audioPath>`：音频路径
- `--language <code>`：语言提示（如 `zh` / `en`）

## 示例

```bash
sma voice models
sma voice init
sma voice on SenseVoiceSmall
sma voice install whisper-large-v3-turbo --force
sma voice use whisper-large-v3-turbo
sma voice transcribe ./demo.ogg --language zh
sma voice off
```

## 说明

安装相关命令会先做本地检查：

- 模型目录已存在时，默认直接跳过下载（`--force` 可强制重下）。
- `voice init` 安装 Python 依赖时，已存在依赖会跳过，并打印 skip 日志。

该命令组通常需要已运行的 daemon：

```bash
sma agent on
```

如果在非交互环境（CI、脚本）使用 `sma voice init`，请显式传入模型参数，例如：

```bash
sma voice init SenseVoiceSmall
```
