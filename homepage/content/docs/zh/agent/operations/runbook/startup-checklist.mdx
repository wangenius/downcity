# 启动前检查清单

在执行 `city agent start` 之前，建议先确认：

1. 项目已经初始化
2. `downcity.json.execution.modelId` 已配置
3. 全局模型池中存在对应模型
4. 如果启用了聊天渠道，对应 `channelAccountId` 确实存在
5. 需要的环境变量已经准备好

其中最容易被忽略的通常是：

- `modelId` 虽然写了，但全局模型池里并不存在
- 环境变量虽然在本地 shell 有，但没有真正出现在运行时环境里
