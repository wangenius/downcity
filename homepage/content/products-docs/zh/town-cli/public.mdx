# Public Console CLI 已移除

当前版本的 `town` 不再提供 Console 公网暴露命令。`town start` 只启动本机 runtime，不会启动或重新绑定 Console UI。

如果你需要让 Console 对外访问，请在 Console UI 的产品部署、反向代理、隧道或云端托管层完成公网暴露。
