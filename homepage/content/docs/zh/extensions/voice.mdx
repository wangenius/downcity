# Voice 扩展

Voice 扩展用于本地语音识别（STT）。

## 常用命令

```bash
city voice models
city voice on SenseVoiceSmall
city voice install whisper-large-v3-turbo
city voice use whisper-large-v3-turbo
city voice transcribe ./demo.ogg --language zh
city voice off
```

## 说明

- `voice on` 可交互选择模型。
- `voice install` 只安装，不改变启用状态；已存在模型会自动跳过（`--force` 强制重下）。
- 当前配置写入 console 全局 `~/.ship/ship.db` 的 `extensions_config.voice`。

## 运行态查看

```bash
city extension status voice
```
