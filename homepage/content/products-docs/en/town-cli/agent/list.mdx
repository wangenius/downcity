# `town agent list`

List agent projects already registered in the managed agent registry.

The registry is the discovery layer for project Agents known to Town.

## Usage

```bash
town agent list
town agent list --running
```

## Behavior

- Shows registered agent projects from the managed agent registry.
- `--running` filters the list to currently active agent daemons only.
- Displays both running and stopped entries.
- Helps you see which agents are available for interactive `town agent start`.

## Registry vs process state

The registry records which project roots Town knows about and which id belongs to each project.

Actual runtime state still depends on daemon inspection. That is why `list` and [`status`](/en/products-docs/town-cli/agent/status) are related, but not interchangeable.

## Related commands

- [`town agent start`](/en/products-docs/town-cli/agent/start)
- [`town agent status`](/en/products-docs/town-cli/agent/status)
