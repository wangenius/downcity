# `town agent doctor`

诊断指定 agent 的状态文件，并可清理僵尸 pid/meta。

> 这页保留为命令入口。更完整的说明见 [CLI / doctor](/zh/docs/agent/cli/maintenance/doctor) 和 [Agent 显示 stale](/zh/docs/agent/operations/troubleshooting/agent-shows-stale)。

## 用法

```bash
town agent doctor [path] [--fix]
```

## 常用参数

- `--fix [enabled]`：发现僵尸状态时自动清理

## 行为说明

- `path` 默认当前目录 `.`。
- 目标 agent 需要已登记在 console registry。
- 不带 `--fix` 时只诊断，不修改文件。
