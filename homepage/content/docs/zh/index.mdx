Downcity 文档聚焦 Studio 本机宿主、City 服务运行时、CLI、配置体系，以及它们如何组成完整的运行逻辑。

## 文档结构

- `quickstart`：安装、初始化、downcity.json
- `chrome-extension`：安装连接、Extension Popup 与页内输入框（Inline Composer）、快捷键、抓取行为、排障
- `configuration`：配置分层、downcity.json、模型配置、上线检查清单
- `security`：权限边界、运行隔离、密钥与数据安全
- `commands`：console、plugin、chat、task、memory、shell、asr、tts
- `concepts`：架构、Execution Context 与 Session、Plugin 协作模型、关系与进程模型、消息链路
- `operations`：日志、故障排查、FAQ
- `reference`：命令参考、HTTP API、`.downcity/` 目录

## 版本说明

本文档以当前仓库 `cli/studio/` 中的 Studio CLI 行为，以及 `packages/city/` 中的 City 服务运行时为准。

如果你要看嵌入式接入、Session、Plugin 或内建能力组合，请继续阅读 [Agent SDK 文档](/zh/agent-sdk-docs)。
