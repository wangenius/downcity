# `sma status`

查看 Agent Runtime daemon 状态。

## 用法

```bash
sma status [path]
sma agent status [path]
```

## 输出规则

- 运行中：显示 `pid`、`log`、`startedAt`
- 已初始化但未运行：显示 `daemon is not running`
- 未初始化：提示执行 `shipmyagent init`

## 示例

```bash
# 查看当前目录
sma status

# 显式指定目录
sma status /path/to/agent

# 与 agent 子命令等价
sma agent status .
```
