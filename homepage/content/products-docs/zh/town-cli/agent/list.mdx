# `town agent list`

列出已经登记到受管 agent registry 的 agent 项目。

registry 是 Town 发现和选择项目 Agent 的本地记录层。

## 用法

```bash
town agent list
town agent list --running
```

## 行为说明

- 展示受管 agent registry 中已登记的 agent 项目。
- `--running` 只保留当前仍在运行的 agent daemon。
- 同时展示运行中和已停止的记录。
- 方便在交互式 `town agent start` 之前先确认有哪些 agent 可选。

## Registry 和进程状态

registry 记录 Town 已知的项目根目录，以及每个项目对应的 agent id。

真实运行状态仍然要靠 daemon 检查确认。所以 `list` 和 [`status`](/zh/products-docs/town-cli/agent/status) 相关，但不能互相替代。

## 相关命令

- [`town agent start`](/zh/products-docs/town-cli/agent/start)
- [`town agent status`](/zh/products-docs/town-cli/agent/status)

## 示例输出

```text
town v1.0.437

Agents                                           2 registered
  lucas_whitman
    project   /Users/you/Documents/bots/lucas_whitman
    status    running
  research_lab
    project   /Users/you/Documents/bots/research_lab
    status    stopped
```
