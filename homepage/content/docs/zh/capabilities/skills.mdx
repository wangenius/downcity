# Skills

Skill 是围绕 `SKILL.md` 组织的可复用指令。

它让 Agent 学会可重复的工作流，而不需要把这些工作流硬编码进核心 runtime。

## 发现目录

Downcity 可以从这些位置发现 skills：

- 项目级 skill roots
- 用户级 skill roots
- skill plugin 配置里声明的额外 roots

项目级 roots 适合只属于某个仓库的工作流。用户级 roots 适合跟随操作者跨项目复用的工作流。

## 运行时语义

Skill 会在执行前被选择，然后注入 Agent context。

这意味着基础 Agent 可以保持通用，但在具体任务需要时获得精确的流程知识。

## 重要边界

已安装 skill 代表可用知识，不代表自动可信的 action。

Agent 仍然需要遵守每个 skill 声明的工具边界，以及当前生效的权限模型。
