# Downcity 文档

Downcity 把代码仓库启动为可对话、可执行、可审计的 Agent Runtime。

## 阅读路线

1. [快速开始](/zh/docs/quickstart/getting-started)
2. [核心原理](/zh/docs/concepts)
3. [配置中心](/zh/docs/configuration/overview)
4. [安全与权限](/zh/docs/security/overview)
5. [Service](/zh/docs/services)
6. [运维与排障](/zh/docs/operations)
7. [CLI / API 参考](/zh/docs/reference)

## 文档结构

- `quickstart`：安装、初始化、ship.json
- `configuration`：配置分层、ship.json、模型配置、上线检查清单
- `security`：权限边界、运行隔离、密钥与数据安全
- `services`：Skills、Tasks、Telegram、飞书、QQ（统一入口）
- `commands`：console、service、plugin、voice
- `concepts`：架构、Service Runtime、Extension Runtime、关系与进程模型、消息链路
- `operations`：日志、故障排查、FAQ
- `reference`：命令参考、HTTP API、`.ship/` 目录

## 版本说明

本文档以当前仓库 `packages/downcity` 中可执行 CLI 和类型定义为准，已清理旧版无效命令与过时页面。
