# Feishu Integration

## Required config

```json
{
  "services": {
    "chat": {
      "channels": {
        "feishu": {
          "enabled": true,
          "appId": "${FEISHU_APP_ID}",
          "appSecret": "${FEISHU_APP_SECRET}",
          "domain": "https://open.feishu.cn"
        }
      }
    }
  }
}
```

## Start runtime

```bash
sma agent on
```

## Verify channel status

```bash
sma service status chat
```
