# QQ Adapter 使用说明（direct 模式）

## 能力范围
- 负责 QQ 官方机器人网关（WebSocket）消息接入与发送。
- 入站消息会由 chat service 映射到内部 `contextId`（随机分配并持久化）。

## 使用约束
- 在 direct 模式下，你输出的 assistant 文本会自动发送到当前 QQ 会话。
- QQ 出站回复依赖入站消息上下文（如 `chatType` 与 `messageId`）；跨会话发送请使用 `sma chat send --chat-key ...`，不要在 direct metadata 中写 `chatKey`。
- 群聊消息默认全量接入，不需要 `@` 触发。
- 适配器会过滤机器人自回环消息并做入站去重，避免“自己回复自己”的循环。
