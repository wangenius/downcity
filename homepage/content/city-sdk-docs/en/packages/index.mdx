# Packages

City SDK docs focus on the `@downcity/city` runtime package and the CLI entry points around Federations and local Agents.

## City SDK Surface

- [@downcity/city](/en/city-sdk-docs/packages/city): City, Federation, service/action infrastructure, hooks, store, table, and the client access layer.
- [CLI](/en/city-sdk-docs/packages/cli): `fed/downfed` manages Federation deploys and admin operations; `city/downcity` manages local Agents and user-side connections.

## Related Spaces

- [Federation Services Docs](/en/payments): official service packages such as accounts, balance, usage, and Stripe payment.
- [Products Docs](/en/products-docs): Downcity CLI, Console, and Chrome Extension product manuals.
- [Agent SDK](/en/agent-sdk-docs): programmable agents, sessions, local/remote agent, and plugin APIs.
