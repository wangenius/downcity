# `town env`

`town env` 用来管理 Console Env 条目。

## 用法

```bash
town env list [--json]
town env set <key> <value> [--description <text>] [--json]
town env copy
town env delete <key> [--json]
```

## 行为说明

- `list` 用来查看当前已经配置了哪些 key。
- `set` 用来新增或更新一个 key。
- `copy` 用 `.env` 文件格式输出明文 value，可直接重定向成 `.env`。
- `delete` 用来删除一个 key。
- 这些 key 的 value 存在 Console store 中，并按加密方式保存。
- `list` 不会返回任何明文 value。
- `copy` 会显式输出明文 value，只在你需要导出 `.env` 时使用。
- `copy` 只能在本机 CLI 中执行，不能从 agent shell 中执行。
- 当前版本只管理 Console 全局 env。

## 导出 `.env`

```bash
town env copy > .env
```

示例输出：

```dotenv
OPENAI_API_KEY=sk-example
PROMPT_PREFIX="value with spaces"
```

## 示例输出

```text
$ town env list
Env                                                2 configured
  OPENAI_API_KEY
    scope         global
    description   Shared OpenAI key
  TELEGRAM_BOT_TOKEN
    scope         global
```
