# 使用说明

## Extension Popup 怎么用

1. 点击浏览器工具栏中的扩展图标
2. 选择当前要使用的 Server Connection
3. 选择目标 Agent
4. 选择目标 Session
5. 输入任务说明
6. 点击发送，或按 `Cmd/Ctrl + Enter`

Popup 适合做明确、完整的一次网页投递。

## 发送时实际发生了什么

扩展会按下面的顺序工作：

1. 读取当前网页标题和 URL
2. 抓取页面正文并转换成 Markdown
3. 生成一个附件文件
4. 把 Ask 和 Markdown 附件一起发送到目标 Agent / Session

## 页面历史有什么用

Popup 会记录当前页面在当前连接下最近的发送历史，方便你：

- 回看之前发过什么
- 避免重复输入
- 确认自己发到了哪个连接下

## 常用快捷键

- `Cmd/Ctrl + Enter`：发送当前 Ask

## 多连接怎么切换

如果你配置了多个 Server Connection，Popup 顶部会显示连接选择器。

切换连接后：

- Bearer Token 会切换成该连接自己的 token
- 默认 Agent / Session 会切换成该连接自己的路由偏好
- 页面历史也会按该连接重新过滤
