# Town Console

Town Console is the browser control surface for Downcity. Current `town` CLI no longer starts Console UI; `town start` only starts the local runtime.

## Main Tasks

- inspect agent status
- manage plugin configuration and Town user City connection state
- check runtime state while debugging

When developing Console UI in this repository, use the workspace Console development entry. Production deployments should expose Console through the product deployment path.
