# `town init`

Initialize Town global runtime infrastructure.

## Usage

```bash
town init
```

## Behavior

- Ensures `~/.downcity/` exists for Town runtime files
- Generates `~/.downcity/schema/downcity.schema.json` (editor schema)

## Notes

- This is a console-level command and does not depend on an agent project path.
- For project-level configuration, use [`town config`](/en/products-docs/town-cli/config).
- `town init` does not create or update models. Models are exposed by the shared AI service.
