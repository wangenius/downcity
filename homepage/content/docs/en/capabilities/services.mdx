# Chat Channels

The chat service supports these channels:

- Telegram
- Feishu / Lark
- QQ Bot

## Channel guides

- [Telegram](/en/docs/services/telegram)
- [Telegram Message Flow](/en/docs/services/telegram-message-flow)
- [Feishu](/en/docs/services/feishu)
- [QQ](/en/docs/services/qq)

## Enable channels

Configure `ship.json.services.chat.channels` and start runtime:

```bash
sma agent on --daemon
```

## Runtime management

```bash
sma service list
sma service status chat
sma service restart chat
```
