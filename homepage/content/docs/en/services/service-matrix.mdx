# Service Matrix

This page lists **actual services registered in runtime**.

## Registered services

From `package/src/main/service/Services.ts`, runtime registers:

- `chat`
- `skill`
- `task`
- `memory`

## Why this matters

- `services/*` should be read as business runtime modules, not only chat channels.
- Channel docs (Telegram/Feishu/QQ) are only one part of the `chat` service.
