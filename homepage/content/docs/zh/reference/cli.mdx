## 新运行逻辑（简版）

- `sma start`：启动 manager（后台）
- `sma agent .`：等价 `sma agent on .`
- `sma agent on .`：默认后台
- `sma agent on . --foreground`：前台
- `sma status .`：查看当前目录 agent 状态
- `sma agent off .`：关闭当前目录 agent
