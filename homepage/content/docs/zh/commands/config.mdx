# `bay config`

管理项目 `downcity.json` 与 alias。

> 命令名说明：配置命令是 `bay config`，不是 `bay console config`。

## 用法

```bash
bay config <subcommand>
```

## 子命令

- `get [keyPath]`
- `set <keyPath> <value>`
- `unset <keyPath>`
- `alias`

> 模型池管理已迁移到独立命令：[`bay model`](/zh/docs/commands/model)。

## Plugin 配置

`bay config` 现在只读写项目 `downcity.json`。

插件相关配置请改为：

- `plugins.*` 只管理当前 agent 的插件参数
- plugin 启停请使用 `bay <plugin> on/off`

示例：

```bash
bay config get plugins.asr
bay config set plugins.asr.modelId '\"SenseVoiceSmall\"'
```

## 示例

```bash
bay config get model.primary
bay config set model.primary fast
bay config unset plugins.chat.channels.telegram
bay config alias --dry-run
```
