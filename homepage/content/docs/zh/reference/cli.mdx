# CLI 命令

CLI 文档按“单命令页面”组织。

## 命令目录

- 直接在 `Commands` 分组中选择：`console` / `agent` / `service` / `extension`

## 运行模型（简版）

- `sma console start`：启动全局控制面
- `sma agent create`：初始化项目运行文件
- `sma agent start`：启动项目 agent daemon
- `sma service ...`：运维核心 service 能力
- `sma extension ...`：运维 extension 扩展能力
- `sma console model ...`：管理运行时模型配置

## 保持最新

本地帮助输出永远是最新的：

```bash
sma --help

```
