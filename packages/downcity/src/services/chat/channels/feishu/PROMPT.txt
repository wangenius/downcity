# Feishu Adapter 使用说明

## 能力范围
- 负责飞书（Lark）消息事件接入与回发。
- 入站消息会由 chat service 映射到内部 `contextId`（随机分配并持久化）。

## 使用约束
- 群聊消息默认全量接入，不需要 `@` 触发。
- 发送时由适配器按 chat 类型自动选择回复模式（如群聊回复到消息、私聊直发）。
- 平台事件可能重复投递，适配器会做去重；业务层无需重复实现去重逻辑。
- 支持在回复正文中使用 `@attach` 发送附件（当前按文件消息发送）：
  - `@attach document <相对路径> | 可选说明`
  - `@attach file <相对路径> | 可选说明`
