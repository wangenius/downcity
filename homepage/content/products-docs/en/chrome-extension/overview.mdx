# Chrome Extension Overview

The Chrome extension is the browser entry point for Downcity Town.

It has two main paths: the Side Panel opens a persistent browser chat with an Agent, while the popup captures the current page as Markdown and sends it to the selected Agent.

## Best Use Cases

- You are reading an article and want to discuss it with an Agent in the browser side panel
- You want to send a live page snapshot to an Agent
- You want one simple browser tool connected to local or remote Downcity Town

## What It Provides

### 1. Side Panel

The Side Panel lets you:

- keep a persistent browser chat open
- connect to the Town runtime API through `/api/sdk/*`
- send regular messages with the current tab attached as an input reference
- quote selected page text from an on-page `引用` popover near the selection
- subscribe to Agent events and see replies in place

### 2. Extension Popup

The popup lets you:

- choose the target Agent
- write an Ask and send it
- review recent page history for the current Town

### 3. Settings Page

The settings page lets you:

- set the `Town URL`
- choose the default Agent
- set a default Ask for the popup
- add a Town Token only when the target Town requires it

## What It No Longer Does

- no Inline Composer
- no on-page input panel
- only a lightweight on-page selection quote popover
- no waiting for the final execution result inside the popup
- no multi-field Server configuration in the settings page
