# 项目初始化

`city agent create` 创建项目运行目录和 Skills 目录，并把 Agent 配置写入全局数据库。

```bash
city agent create my-project
```

生成结构：

```text
my-project/
├── .env
├── .agents/
│   └── skills/
└── .downcity/
```

全局数据库保存 Agent ID、模型绑定、启动参数和 Plugin 配置。项目 `.env` 会覆盖平台全局环境和当前 CLI 进程环境。

```bash
# 允许覆盖全局数据库中的现有 Agent 配置
city agent create my-project --force
```

初始化后，可在 `.agents/skills` 添加项目 Skills，并运行 `city agent start` 启动 Agent。
