# CLI Commands

## Agent lifecycle

```bash
# Create a new project
downcity agent init my-project

# Start the agent in the current directory
downcity agent start

# Stop the running agent
downcity agent stop

# Restart the agent
downcity agent restart

# Check status
downcity agent status

# View logs
downcity agent logs
```

## City identity

```bash
# Login to City
downcity login

# Logout
downcity logout

# Check who you are
downcity whoami

# List API tokens
downcity tokens list

# Create a new token
downcity tokens create --name "ci-deployment"

# Revoke a token
downcity tokens revoke <token-id>
```

## City usage & billing

```bash
# View current usage
downcity usage

# View balance
downcity balance

# Add balance ( Stripe )
downcity balance add --amount 100
```

## Channel accounts

```bash
# Add a Telegram bot
downcity chat add telegram --id telegram-main

# Add a Feishu bot
downcity chat add feishu --id feishu-work

# List accounts
downcity chat list

# Remove an account
downcity chat remove telegram-main
```

## Federation models

```bash
# List available models
downcity models list

# Show model details
downcity models show quality

# List providers
downcity providers list

# Check provider health
downcity providers status
```

## Global flags

Most commands support:

- `--config <path>` — use a custom config file
- `--verbose` — print detailed output
- `--json` — output in JSON format
