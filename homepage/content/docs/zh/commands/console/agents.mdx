# `sma console agents`

列出 console 托管的 agent（包含运行中与历史记录）。

## 用法

```bash
sma console agents [--json]
```

## 行为说明

- 列出当前运行中的 agent runtime。
- 历史记录会保留在 registry 中（`status=stopped`）。
- `--json` 输出结构化结果。

> Console UI 的 `Global Agents` 面板可直接从历史记录里点击“启动”重新拉起 agent。
