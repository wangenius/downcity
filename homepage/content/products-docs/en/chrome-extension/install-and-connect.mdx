# Install and Connect

## Prerequisites

Before using the extension, start Town and an Agent:

```bash
town start
town agent start
```

If you load the extension from this repo, build it first:

```bash
npm run build:extension
```

## Load It in Chrome

1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Choose `products/chrome-extension/dist`

## First-Time Configuration

Open the extension settings page and keep the setup simple:

1. Set `Town URL`
2. Choose `Agent`
3. Set `Default Ask`
4. Click `保存并检查`

For local development, the Town URL is:

```txt
http://127.0.0.1:5314
```

Do not use `15314`. That is the Agent RPC port, not the browser HTTP endpoint.

## Town Token

If the target Town requires auth, the settings page shows `Town Token`.

Create a token in the local terminal:

```bash
town token create my-token
```

If you prefer the interactive flow, run:

```bash
town token
```

Then paste the returned token into `Town Token` and click `保存并检查`.

## Remote Town

For a remote deployment, enter its public HTTPS URL as `Town URL`.

Make sure the URL is reachable from Chrome and points to the Town runtime HTTP API.

## After Updating the Extension

Whenever extension code changes, rebuild it:

```bash
npm run build:extension
```

Then go back to `chrome://extensions` and press the refresh button for the unpacked extension.
