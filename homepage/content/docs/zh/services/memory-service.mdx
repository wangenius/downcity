# Memory Service

`memory` service 提供运行时记忆操作能力。

## 职责

- 状态查看与索引重建
- 记忆检索与按文件读取
- 显式写入 memory
- 把会话窗口刷写到 daily memory

## 运行态操作

```bash
sma service status memory
sma service command memory search --payload '{"query":"release decision"}'
```
