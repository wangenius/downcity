---
title: AI Service
description: AIService 管理 AI 模型，提供两条独立通路：SDK 通路和 OpenAI 兼容通路。
---

`AIService` 是内建的 Service，通过 Provider 注册模型，为每个模型提供两条调用通路。

## 两条通路

| | SDK 通路 | OpenAI 兼容通路 |
|---|---|---|
| 路由 | `POST /v1/ai/text`, `/stream`, `/image`, ... | `POST /v1/ai/chat/completions` |
| 调用方 | `User City`、Agent 的 `CityModel` | OpenAI SDK、curl |
| 输入格式 | text action 或 LanguageModelV3 transport | `{ messages, model, stream }` |
| 返回格式 | UIMessage 或 LanguageModelV3 SSE | OpenAI 格式 JSON / SSE |
| Provider 能力 | `text` / `stream(ctx, call)` | `openai`（未提供则自动透传） |

## 快速开始

```ts
import { createOpenAICompatible } from "@ai-sdk/openai-compatible";
import { AIService, Provider, type OpenAICompatibleClientConfig } from "@downcity/city";

class DeepSeekProvider extends Provider {
  constructor() {
    super({
      id: "deepseek",
      baseURL: "https://api.deepseek.com/v1",
      envKey: "DEEPSEEK_API_KEY",
    });
  }

  protected createClient({ apiKey, baseURL }: OpenAICompatibleClientConfig) {
    return createOpenAICompatible({ apiKey, baseURL, name: "deepseek" });
  }
}

const deepseek = new DeepSeekProvider();

// 注册模型
const ai = new AIService();
ai.use(deepseek.model({ id: "deepseek-v4-flash", name: "DeepSeek V4 Flash" }));
base.use(ai);
```

注册后自动生成路由：

```
POST /v1/ai/text              → SDK 文本
POST /v1/ai/stream            → CityModel LanguageModelV3 模型流
POST /v1/ai/chat/completions  → OpenAI 兼容（自动透传）
```
