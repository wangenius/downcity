# 快速开始

## 1. 安装

```bash
npm install -g downcity
downcity --version
```

> `downcity` 与 `city` 都是同一个 CLI 可执行命令。

## 2. 初始化 Console（仅首次）

```bash
city console init
```

该命令会生成 Console 全局配置（位于 `~/.ship/`）。

## 3. 启动 Console

```bash
city console start
```

> `city agent start` 依赖 console 先运行。

## 4. 初始化项目

```bash
cd /path/to/your-repo
city agent create .
```

初始化后会生成：

- `PROFILE.md`（必需）
- `SOUL.md` / `USER.md`（可选）
- `ship.json`
- `.ship/` 运行目录

## 5. 配置 LLM

在项目根目录创建 `.env`（不要提交到 Git）：

```bash
LLM_API_KEY=your_key
```

`ship.json` 最小示例：

```json
{
  "name": "my-agent",
  "version": "1.0.0",
  "model": {
    "primary": "default"
  }
}
```

## 6. 启动 Agent 运行时

启动（默认后台 daemon，适合长期运行）：

```bash
city agent start
```

需要在当前终端前台观察日志时再用：

```bash
city agent start --foreground
```

## 7. 验证运行状态

```bash
city agent status
city console agents
```

说明：

- agent 端口由 console 自动分配，不建议在快速开始里固定写死端口。
- 多 agent 场景下，每个 agent 端口可能不同。

## 8. 可选：启动 Console UI

```bash
city console ui start
```

## 9. 可选：Chrome 扩展网页选区直发 Agent

如果你在使用仓库内的 `chrome-extension`：

1. 在网页中选中文本后，选区右下角会出现消息按钮，点击后输入框会在选区左下角展开。
2. 点击浏览器扩展图标会打开插件 popup；按 `Cmd/Ctrl + U` 可直接打开页面内输入框。
3. popup 支持 `Ask 历史` 下拉，可快速回填最近提问。
4. 不再额外展示引用 `tag`，当前选区会直接作为发送上下文。
5. 输入 `/` 可快速选择历史 ask 记录。
6. 输入需求后按 `Cmd/Ctrl + Enter` 或点击发送按钮提交（`Enter` 换行）。
7. 若当前没有选区，发送时会自动按页面全文模式投递。
6. 按 `Esc` 可关闭输入框。
