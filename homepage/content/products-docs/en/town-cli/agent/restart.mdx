# `town agent restart`

Restart the process for a target agent.

Use this command after changing startup-loaded project configuration.

## Usage

```bash
town agent restart [path] [options]
```

## Key options

- `-h, --host <host>`: daemon host

## Behavior

- `path` defaults to current directory `.`
- Target agent must be registered in the managed agent registry
- Restart flow is stop first, then start again with a new Town runtime-assigned port
