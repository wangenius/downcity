# Chat Is Not Responding

Check these first:

1. `city agent status`
2. whether the target agent is actually `running`
3. whether model binding is valid
4. if this is a channel message, whether the channel binding is valid

## CLI scenario

If `city agent chat` does not work, first verify that the target agent is really running.

## Channel scenario

If Telegram, Feishu, or QQ is not responding, check `channelAccountId` first.

Many "chat is not responding" issues are not about the message content at all. They usually come from:

- the target agent not running
- invalid model binding
- dangling channel binding
