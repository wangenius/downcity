# `town init`

初始化全局 console 基础设施。

## 用法

```bash
town init
```

## 行为

- 写入/更新 `~/.downcity/downcity.db`（全局 console 基础设施）
- 生成 `~/.downcity/schema/downcity.schema.json`（编辑器 schema）

## 说明

- 这是 console 级命令，不依赖 agent 项目路径。
- 后续项目级配置请使用 [`town config`](/zh/products-docs/town-cli/config)。
- `town init` 不会创建或更新模型。
- 后续可通过 `town model create` 配置模型。

$ town init
