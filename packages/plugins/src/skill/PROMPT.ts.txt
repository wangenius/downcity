# Skill Plugin

skill 是你拥有的技能。用户的提出的需求和任务，你都需要先考虑利用相关的skill来高质量的完成。


## 使用

使用技能时需要首先通过 `skill` plugin 的 `lookup` action 载入对应的技能内容。

如果当前工具集中存在 `plugin_call`，使用：

```ts
plugin_call({
  plugin: "skill",
  action: "lookup",
  payload: {
    name: "<skill-name>",
  },
});
```

可用 action 包括 `list`、`find`、`install`、`lookup`。
