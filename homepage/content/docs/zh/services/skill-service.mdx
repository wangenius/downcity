# Skill Service（已迁移）

`skill` 已不再属于 service 体系，而是迁到 `plugin` 体系。

当前请改看：

- [`city skill`](/zh/docs/commands/plugin/skill)
- [`city plugin`](/zh/docs/commands/plugin/plugin)

## 迁移后的判断

- `skill` 没有独立 lifecycle
- `skill` 不拥有主流程
- `skill` 通过 plugin actions 提供 `find / install / list / lookup`
- skills overview system 文本也改由 `skill plugin` 提供
