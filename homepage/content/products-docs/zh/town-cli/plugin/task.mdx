# `town task`

`town task` 是托管 task plugin 的命令组。

当你要定义“可复用、可长期存在的自动化任务”时，优先使用它，而不是一次性的延迟 plugin action。

## 用法

```bash
town task <subcommand> [options]
```

## 主要子命令

- `list`
- `create`
- `run <title>`
- `update <title>`
- `delete <title>`
- `status <title> <enabled|paused|disabled>`
- `enable <title>`
- `disable <title>`

## 边界

- `town task` 适合可复用、长期存在的自动化。
- `town chat send --delay/--time`、`town memory remember --delay/--time` 适合“把现成 action 稍后执行一次”。
- 任务定义可以使用 `@manual`、cron，或 `time:<ISO8601>`。

## 示例

```bash
town task list
town task create --title "Daily check" --description "Run tests" --session-id telegram-chat-123 --when "0 10 * * 1-5" --activate
town task run daily-check
town task update daily-check --when "@manual"
town task disable daily-check
```
