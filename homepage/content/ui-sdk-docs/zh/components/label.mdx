# Label

`Label` 用于给输入控件提供明确命名。

## 导入

```tsx
import { Label } from "@downcity/ui";
```

## 最小示例

```tsx
import { Input, Label } from "@downcity/ui";

export function NameField() {
  return (
    <div className="flex flex-col gap-2">
      <Label htmlFor="agent-name">Agent 名称</Label>
      <Input id="agent-name" />
    </div>
  );
}
```

## 使用建议

- 总是把 `htmlFor` 指向对应控件的 `id`
- 勾选项时和 `Checkbox` 放在同一行
- `Label` 负责语义，不负责复杂布局
