# Task 插件

Task 插件让 Agent 能够调度和运行后台任务。它适用于定期数据获取、报告生成，或任何应该在正常请求-响应流之外发生的工作。

## 功能

- 在特定时间或间隔调度任务
- 在 Agent 重启后持久化任务状态
- 处理重试和失败恢复
- 提供延迟任务队列

## 定义任务

任务在项目定义中，并在启动时向 Task 插件注册。它们可以通过以下方式触发：

- Cron 表达式
- 一次性调度
- 事件驱动触发

## 使用场景

- 每日数据同步
- 定期报告生成
- 后台文件处理
- 定时通知
