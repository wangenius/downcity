# CLI 命令

Downcity docs 解释设计模型、运行边界和配置语义。

统一的 `downcity` 命令（别名 `city`）既负责本机 Agent 宿主，也负责 City 管理。精确参数以本机安装的 CLI help 为准；City 侧能力详见 City SDK 文档。

- [City CLI](/zh/city-sdk-docs/packages/cli)

## 本地帮助

精确参数以本机安装的 CLI help 为准。产品文档负责说明命令组和常见工作流。

## 交互式 TUI

不带子命令运行 `city` 或 `downcity` 时，现在默认进入交互式 TUI。`city` 会先显示 City 列表，选择某个 City 后直接进入该 City 的 admin 管理；两个 CLI 都保持同一套稳定布局：左侧 sidebar 承载菜单层级，顶部显示 breadcrumb，loading、输入表单、表格、JSON、消息与选择详情都在右侧 section 内完成，直到你主动退出 TUI。

## CLI 语言切换

`city` 和 `downcity` 现在都支持中文与英文。

- 通过 `--lang zh` 或 `--lang en` 显式切换当前命令语言。
- 如果没有传 `--lang`，CLI 会优先读取最近一次在交互菜单中保存的默认语言，然后再依次读取 `DC_CLI_LANG`、`LC_ALL`、`LC_MESSAGES`、`LANG`。
- 在交互模式下，`city` 和 `downcity` 都可以直接从菜单切换语言，并把该选择持久化到本地。

示例：

```bash
downcity --lang en --help
city --lang zh
DC_CLI_LANG=zh downcity agent --help
```
