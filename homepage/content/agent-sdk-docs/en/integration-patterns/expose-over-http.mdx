# Expose Agent over HTTP

If the caller and the executor do not live in the same process, you can expose the local Agent as an HTTP service.

In Downcity, HTTP exposure is published by Town:

```bash
town agent start
```

For local process-to-process access, prefer `rpc://`. Use HTTP when an external client needs to call the Agent through the Town gateway.
