# `sma agent restart`

重启指定 agent 的 runtime。

## 用法

```bash
sma agent restart [path] [options]
```

## 常用参数

- `-p, --port <port>`：服务端口
- `-h, --host <host>`：服务主机

## 行为说明

- `path` 默认当前目录 `.`。
- 目标 agent 需要已登记在 console registry。
- 流程是先停后起，最终按新参数启动。
