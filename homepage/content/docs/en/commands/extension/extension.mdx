# `sma extension`

Manage extension runtime state and forward extension commands.

## Command hierarchy

- Top-level command: `sma extension`
- Action command: `sma voice`

`sma voice` is an extension action command, not a top-level command.

## Usage

```bash
sma extension <subcommand>
```

## Subcommands

- `list`
- `status <extensionName>`
- `start <extensionName>`
- `stop <extensionName>`
- `restart <extensionName>`
- `command <extensionName> <command> --payload <json>`

## Examples

```bash
sma extension list
sma extension status voice
sma extension restart voice
sma extension command voice status --payload '{}'
```

## Notes

Start agent runtime first:

```bash
sma agent start
```
