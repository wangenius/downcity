# `sma console`

`console` 是全局控制面。

你可以通过它管理 console 运行状态，并查看当前托管的 agent runtime。

## 命令层级

一级命令固定为 4 个：

- `sma console`
- `sma agent`
- `sma service`
- `sma extension`

`console` 下面再细分 `init/model/config/start/stop/status/agents/ui`。

## 用法

```bash
sma console init
sma console model <subcommand>
sma console config <subcommand>
sma console start
sma console stop
sma console restart
sma console status
sma console agents [--json]
sma console ui [start|stop|restart|status] [--port 3001]
```

## 行为说明

- `init`：初始化全局 console 配置（`~/.ship/*`）。
- `model`：独立管理 console 全局语言模型池（provider/model/test/pause）。
- `config`：管理项目级 `ship.json` 与 alias。
- `start`：后台启动 console。
- `stop`：停止 console，并停止其托管的子 agent runtime。
- `restart`：重启 console。
- `status`：查看 console 进程状态与活跃 agent 清单。
- `agents`：打印当前活跃托管 agent 列表。
- `ui`：管理单实例 UI（start/stop/restart/status），并可在页面内切换查看不同 agent。

## 典型流程

```bash
# 1) 启动全局 console
sma console start

# 2) 启动项目 runtime
cd /path/to/agent-project
sma agent create .
sma agent on

# 3) 查看状态
sma console status
sma agent status .
```
