# `sma agent`

Manage Agent Runtime start/stop/restart.

## Usage

```bash
sma agent [path]
sma agent on [path] [options]
sma agent off [path]
sma agent status [path]
sma agent restart [path] [options]
```

## Subcommands

- `on`: start runtime
- `off`: stop daemon runtime
- `status`: show daemon runtime status
- `restart`: restart daemon runtime

## Options

### `on` options

- `-p, --port <port>`: service port (or `ship.json.start.port`)
- `-h, --host <host>`: service host (or `ship.json.start.host`)
- `--webui [enabled]`: enable interactive Web UI (or `ship.json.start.webui`)
- `--webport <port>`: Web UI port (or `ship.json.start.webport`)
- `--foreground [enabled]`: run in foreground (current terminal)
- `--daemon [enabled]`: backward-compatible daemon flag (legacy form)

### `restart` options

- `-p, --port <port>`: service port (or `ship.json.start.port`)
- `-h, --host <host>`: service host (or `ship.json.start.host`)
- `--webui [enabled]`: enable interactive Web UI (or `ship.json.start.webui`)
- `--webport <port>`: Web UI port (or `ship.json.start.webport`)

## New runtime logic

- `sma agent .`: alias of `sma agent on .`
- `sma agent on .`: default is background daemon mode
- `sma agent on . --foreground`: foreground mode
- `sma agent off .`: stop daemon for the current directory agent
- If `path` is omitted, default is `.`

## Environment variable injection

On every `sma` execution, CLI resolves current agent context and injects variables into the `sma` process and its child processes:

- `SMA_AGENT_PATH`: absolute path of current agent
- `SMA_AGENT_NAME`: agent name (prefer `ship.json.name`, fallback to directory name)

> Note: this is process-scoped injection and does not mutate your parent shell session.

## Examples

```bash
# start current directory agent in background (default)
sma agent .

# explicit background start
sma agent on .

# foreground start
sma agent on . --foreground

# stop current directory agent
sma agent off .

# show current directory agent status
sma agent status .

# restart current directory agent
sma agent restart .
```
