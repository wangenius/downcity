# `.ship/` 目录

`.ship/` 是 ShipMyAgent 的本地运行数据目录，默认建议不提交到 Git。

## 常见结构

```text
.ship/
  logs/
  context/
  memory/
  channel/
  chat/
  task/
  config/
  schema/
  public/
  profile/
  .cache/
```

## 重点目录

### `.ship/logs/`

每日 JSONL 日志：`<YYYY-MM-DD>.jsonl`。

### `.ship/context/<contextId>/messages/`

会话消息事实源：`messages.jsonl`，并包含 compact 元数据与归档。

### `.ship/memory/`

Memory V2 存储层：
- `MEMORY.md`：长期事实/偏好记忆。
- `daily/<YYYY-MM-DD>.md`：按天追加的记忆流水。
- `index.sqlite`：本地 FTS 索引（可重建）。

### `.ship/chat/`

chat 元信息与 history 事件流。

### `.ship/channel/meta.json`

渠道目标到 `contextId` 的映射快照（由 chat service 维护）。

### `.ship/task/`

任务定义与每次运行产物（`run.json`、`result.md` 等）。

### Skills 目录位置

skills 目录位于 `.agents/skills`（项目级）与 `~/.agents/skills`（用户级）。

## 安全建议

- `.ship/` 可能含敏感对话与执行痕迹，不建议提交到公开仓库。
- 需要排障时优先查看 `logs`、`task`、`context` 三处。
