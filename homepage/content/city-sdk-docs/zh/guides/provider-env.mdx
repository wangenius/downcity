---
title: AIChannel 环境变量
description: 在 Federation 环境中配置上游 API Key。
---

在 Channel 上声明密钥：

```ts
class DeepSeekChannel extends AIChannel {
  constructor() {
    super({
      id: "deepseek",
      env_key: "DEEPSEEK_API_KEY",
      base_url: "https://api.deepseek.com/v1",
    });
  }

  protected async stream(ctx: Context, call: LanguageModelV3CallOptions) {
    const api_key = read_required_env(ctx, this.env_key ?? "");
    // 创建真实上游模型并返回 V3 stream
  }
}
```

`env_key` 会自动为模型目录的管理端视图生成必填 env 声明。一个 Channel 需要多个环境变量时使用 `env`。

值存储在 Federation env 表中，执行时通过 `ctx.env(key)` 或 `read_required_env(ctx, key)` 读取。它们不会进入 `CityModelDescriptor` 或客户端运行时。
