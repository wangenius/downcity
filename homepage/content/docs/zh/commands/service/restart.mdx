# `sma service restart`

重启目标 agent 下的指定 service。

## 用法

```bash
sma service restart <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
sma service restart chat --agent my-agent
```
