# 例子：最小本地 Agent

```ts
import { Agent } from "@downcity/agent";

const agent = new Agent({
  id: "repo-helper",
  path: "/path/to/project",
  tools: {},
});
```

然后：

```ts
const session = await agent.session();
await session.set({ model });
const turn = await session.prompt({ query: "继续" });
await turn.finished;
```

这条最小例子适合先验证 3 件事：

- Agent 能不能成功构造
- Session 能不能成功创建
- 模型能不能成功执行
