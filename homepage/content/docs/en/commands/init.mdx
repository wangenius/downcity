# `city init`

Initialize global console configuration.

## Usage

```bash
city init [--force]
```

## Behavior

- Writes/updates `~/.downcity/downcity.db` (global config + model pool)
- Generates `~/.downcity/schema/downcity.schema.json` (editor schema)

## Key option

- `--force`: overwrite existing global config (dangerous)

## Notes

- This is a console-level command and does not depend on an agent project path.
- For project-level configuration, use [`city config`](/en/docs/commands/config).
