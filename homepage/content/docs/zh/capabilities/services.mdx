# Chat Channels

当前 chat service 支持以下消息渠道：

- Telegram
- 飞书（Feishu/Lark）
- QQ Bot

## 入口文档

- [Telegram](/zh/docs/services/telegram)
- [Telegram 消息链路](/zh/docs/services/telegram-message-flow)
- [飞书](/zh/docs/services/feishu)
- [QQ](/zh/docs/services/qq)

## 生效方式

在 `ship.json.services.chat.channels` 启用对应渠道后，启动 runtime：

```bash
sma agent on --daemon
```

## 运行态管理

```bash
sma service list
sma service status chat
sma service restart chat
```
