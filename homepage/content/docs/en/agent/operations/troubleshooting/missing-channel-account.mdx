# Missing channel account

If the project references a `channelAccountId` that does not exist, the chat channel will not work correctly.

## What to verify

- whether the ID in `downcity.json` is spelled correctly
- whether that account really exists at the global level
- whether the agent was restarted after the change

If a team maintains several agents long-term, standardizing account ID naming early reduces troubleshooting cost a lot.
