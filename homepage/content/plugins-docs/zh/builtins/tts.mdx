# tts Plugin

`tts` 是文本转语音 plugin。它只定义 agent 侧协议，不绑定本地模型、Python 依赖或 provider。

真实语音合成能力从 constructor 传入。推荐把 City 的 AI 能力接进来：

```ts
import { Agent } from "@downcity/agent";
import { TtsPlugin } from "@downcity/plugins";

const agent = new Agent({
  id: "speech-agent",
  path: "/path/to/project",
  plugins: [
    new TtsPlugin({
      tts: (input) => city.ai.tts(input),
      language: "auto",
      format: "wav",
    }),
  ],
});
```

## 设计边界

`tts` 负责：

- 暴露 `synthesize` action
- 调用 constructor 注入的 `tts` 函数
- 把返回结果统一成 AI SDK UIMessage
- 让音频 file part 进入 agent 统一资源落盘链路
- 给 agent 注入如何使用 `tts` 的 system 提示

`tts` 不负责：

- 安装本地模型
- 选择 TTS provider
- 管理 Python 依赖
- 持久化项目配置
- 提供 CLI 管理命令

## Constructor 参数

| 参数 | 作用 | 默认值 |
| --- | --- | --- |
| `tts` | 必填，真实语音合成函数 | 无 |
| `language` | 默认语言提示，例如 `auto`、`zh`、`en` | 无 |
| `voice` | 默认音色提示 | 无 |
| `format` | 默认输出格式提示，例如 `wav`、`mp3`、`ogg` | 无 |
| `name` | plugin 名称 | `tts` |
| `title` | plugin 展示标题 | `TTS` |
| `description` | plugin 说明 | 内建说明 |

## 显式调用

```ts
const result = await agent.plugins.runAction({
  plugin: "tts",
  action: "synthesize",
  payload: {
    text: "你好，欢迎来到 Downcity",
    language: "zh",
    format: "wav",
  },
});
```

payload 中 `text` 必填。其他常用字段包括：

- `language`
- `voice`
- `format`
- `speed`
- `provider_options`

## 返回结果

注入的 `tts` 函数可以直接返回 AI SDK UIMessage，也可以返回简单音频结果：

```ts
return {
  data_url: "data:audio/wav;base64,...",
  media_type: "audio/wav",
  filename: "speech.wav",
};
```

plugin 会统一转成包含 `file` part 的 UIMessage。通过 `plugin_call` 调用时，agent 会把音频落盘到 `.downcity/resources`，并在最终 assistant 消息中保留 `.downcity/resources/...` 这类 Agent 根目录相对路径。

## Agent 如何调用

```ts
plugin_call({
  plugin: "tts",
  action: "synthesize",
  payload: {
    text: "...",
  },
});
```

TTS 是输出生成能力，不会自动拦截消息链路。只有用户明确需要语音、播报或音频文件时才调用。
