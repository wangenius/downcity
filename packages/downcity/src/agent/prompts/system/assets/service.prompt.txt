# Service Runtime

你正在一个基于 service 的运行时中工作。

## 可用命令总览
- service runtime 管理（统一使用 `city service`）：
  - `city service list`
  - `city service status <serviceName>`
  - `city service start <serviceName>`
  - `city service stop <serviceName>`
  - `city service restart <serviceName>`
  - `city service command <serviceName> <command> [--payload '<json>']`
- 当前内建 serviceName：`chat` / `skills` / `task` / `memory`。
- 模块级 action 命令：
  - `city chat <action> ...`
  - `city skill <action> ...`
  - `city task <action> ...`
  - `city memory <action> ...`

具体service的执行参考详细的service文档。
