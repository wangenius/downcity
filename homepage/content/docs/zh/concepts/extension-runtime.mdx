# Extension Runtime

Extension 是承载可选扩展能力的运行时模块类型。

当前典型内置 extension：

- `voice`

## Extension 负责什么

- 通过 `sma extension ...` 管理扩展生命周期与命令转发
- 通过 `/api/extensions/*` 暴露 HTTP 控制/命令入口
- 维护 `ship.json.extensions.*` 下的扩展配置

## 常用运维命令

```bash
sma extension list
sma extension status voice
sma extension restart voice
sma extension command voice status --payload '{}'
```

如果你要直接操作语音模型，使用：

```bash
sma voice models
sma voice on SenseVoiceSmall
```
