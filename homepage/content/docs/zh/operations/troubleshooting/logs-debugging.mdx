# 日志与调试

## 1) 看运行日志

```bash
tail -f .ship/logs/$(date +%F).jsonl
```

## 2) 看服务状态

```bash
sma service list
sma extension list
```

## 3) 看会话数据

```bash
sma chat context --chat-key ctx_xxxxxxxxxxxxxxxx
sma chat history --chat-key ctx_xxxxxxxxxxxxxxxx --limit 50
```

## 4) 看任务执行产物

```bash
ls -la .ship/task/<taskId>
ls -la .ship/task/<taskId>/<timestamp>
```

`run.json` + `result.md` 基本可以还原一次任务执行结果。
