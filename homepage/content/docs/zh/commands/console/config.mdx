# `city console config`

管理项目 `downcity.json` 与 alias。

> 命令名说明：配置命令是 `city console config`，不是 `city config`。

## 用法

```bash
city console config <subcommand>
```

## 子命令

- `get [keyPath]`
- `set <keyPath> <value>`
- `unset <keyPath>`
- `alias`

> 模型池管理已迁移到独立命令：[`city console model`](/zh/docs/commands/console/model)。

## Plugin 配置

`city console config` 现在只读写项目 `downcity.json`。

插件相关配置请改为：

- `plugins.*` 管理插件行为
- `assets.*` 管理插件依赖资源

示例：

```bash
city console config get plugins.voice
city console config set plugins.voice.enabled true
city console config set 'assets.voice.transcriber.modelId' '\"SenseVoiceSmall\"'
```

## 示例

```bash
city console config get model.primary
city console config set model.primary fast
city console config unset services.chat.channels.telegram
city console config alias --dry-run
```
