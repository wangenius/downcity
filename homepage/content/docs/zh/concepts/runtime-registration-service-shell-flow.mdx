# Console 注册、Runtime 执行与 Shell 流程

## 1. Console 注册

- registry 文件：`~/.ship/console/agents.json`
- 保存已知 agent 与最近一次 daemon 元数据
- daemon 启动后必须成功写 registry，否则回滚

## 2. Runtime 执行

- 一个 agent 进程绑定一个 `rootPath`
- runtime 组装 services、plugins、capabilities、assets
- service 在需要时调用 capability

## 3. Shell 流程

- shell 是会话式，不是一次性 exec
- 默认工作目录是当前项目根目录
- 子进程会注入 `DC_CTX_*` 环境变量
