# `city start`

后台启动 city runtime。

启动时会初始化 city runtime 的 registry 和进程状态。
如果当前还没有任何统一账户用户，启动过程会提示设置首个管理员密码；留空时默认使用 `downcity`，用户名固定为 `admin`。

如果你希望一并启动 Console，请使用 `city start --console` 或 `city start -a`。

## 用法

```bash
city start
```

```bash
city start --console
```
