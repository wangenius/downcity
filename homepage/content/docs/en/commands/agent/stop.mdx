# `town agent stop`

Stop the background daemon for a target agent.

> This page remains as a command entry point. For the fuller explanation, see [CLI / stop](/en/docs/agent/cli/lifecycle/stop).

## Usage

```bash
town agent stop [path]
```

## Behavior

- `path` defaults to current directory `.`
- Target agent must be registered in the city registry
- Stops the matching daemon process group
- Cleans daemon pid/meta files
- Marks the agent as `stopped` in the city registry
- If the daemon is not running, the command exits successfully with a not-running message
