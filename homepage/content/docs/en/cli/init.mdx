# Project Init

`city agent create` creates the runtime and Skills directories, then stores Agent config in the global database.

```bash
city agent create my-project
```

Generated structure:

```text
my-project/
├── .env
├── .agents/
│   └── skills/
└── .downcity/
```

The global database stores the Agent ID, model binding, startup options, and Plugin config. Project `.env` overrides platform global env and the current CLI process env.

```bash
# Allow replacing existing Agent config in the global database
city agent create my-project --force
```

After initialization, add project Skills under `.agents/skills` and run `city agent start`.
