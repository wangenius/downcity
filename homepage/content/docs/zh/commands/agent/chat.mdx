# `city agent chat`

在终端中与指定 agent 做持续性对话。

该命令默认复用和 Console UI 相同的主会话：`consoleui-chat-main`。

## 用法

```bash
city agent chat [--to <agentName>]
```

## 常用参数

- `-t, --to <name>`：目标 agent 名称；省略时会从当前运行中的 agent 里交互选择
- `--host <host>` / `--port <port>`：强制走远程 HTTP transport
- `--token <token>`：远程 HTTP 模式下显式传入 Bearer token

## 行为说明

- 需要在交互式终端中运行。
- 目标 agent 必须已经处于运行中。
- 终端里输入的每一行都会进入同一个共享 Console 主会话。
- 输入 `/exit` 或 `/quit` 退出持续对话。
