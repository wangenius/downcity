# Packages 包

City SDK 文档只保留 `@downcity/city` runtime package，以及管理已部署 City runtime 时需要的工具入口。

## City SDK 能力面

- [@downcity/city](/zh/city-sdk-docs/packages/city)：City、CityBase、service/action 基建、hooks、store、table 和 client 访问层。
- [City CLI](/zh/city-sdk-docs/packages/cli)：面向已部署 City runtime 的运维命令。

## 相关 Space

- [City Services Docs](/zh/services-sdk-docs)：accounts、balance、usage、Stripe payment 等官方 service 包。
- [Products Docs](/zh/products-docs)：Town CLI、Town Console、Chrome Extension 产品手册。
- [Agent SDK](/zh/agent-sdk-docs)：可编程 Agent、Session、Local/Remote Agent 和 Plugin API。
