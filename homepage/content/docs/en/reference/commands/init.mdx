# `sma init`

Initialize a ShipMyAgent project in the target path.

## Usage

```bash
sma init [path]
```

## Arguments

- `path` (optional): project path. Defaults to current directory.

## Example

```bash
# Initialize in current directory
sma init

# Initialize in another directory
sma init ./my-agent-project
```
