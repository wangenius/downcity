# 配置总览

当前推荐配置分层：

1. Console 全局层：管理共享资源（例如模型池）。
2. Agent 项目层：声明当前项目绑定与运行参数。

## 关键原则

1. 模型定义在 Console，全局复用。
2. Agent 只声明 `model.primary`，不重复保存 provider 细节。
3. 任何配置变更后，建议重启对应 agent 验证生效。

## 你会常改的两类文件

1. `~/.ship/ship.db`（通过 CLI/UI 管理，不直接手改）
2. `<project>/ship.json`
