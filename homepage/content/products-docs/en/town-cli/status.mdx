# `town status`

Show Town runtime, Town user City connection, and managed active agents. Current Town CLI no longer reports Console UI runtime state.

## Usage

```bash
town status
```

## Related commands

- [`town agent list --running`](/en/products-docs/town-cli/agent/list)
- [`town city status`](/en/products-docs/town-cli/city)

## Output Sections

- `Town runtime`: running, stale, or stopped.
- `City connection`: current City base, town id, and user token state.
- `Managed agents`: agent daemons that are still running in the managed registry.
