# 恢复聊天运行时

如果 agent 本身能启动，但聊天入口没有反应，优先检查：

1. agent 是否是 `running`
2. 渠道是否启用
3. `channelAccountId` 是否有效
4. 修改后是否已经重启 agent

常见恢复顺序：

```bash
city agent status
city agent restart
```

如果还是不通，再继续检查：

- 模型绑定
- `channelAccountId`
- 消息是否真的进到了目标 agent
