# `bay agent list`

List agent projects already registered in the city registry.

> This page remains as a command entry point. For the fuller explanation, see [CLI / list](/en/docs/agent/cli/lifecycle/list) and [Agent Registry](/en/docs/agent/cli/maintenance/registry).

## Usage

```bash
bay agent list
bay agent list --running
```

## Behavior

- Shows registered agent projects from the city registry.
- `--running` filters the list to currently active agent daemons only.
- Displays both running and stopped entries.
- Helps you see which agents are available for interactive `bay agent start`.

## Related commands

- [`bay agent start`](/en/docs/commands/agent/start)
- [`bay agent status`](/en/docs/commands/agent/status)
