# 使用说明

## Side Panel 怎么用

1. 点击浏览器工具栏中的扩展图标
2. 直接输入消息并发送

Side Panel 使用设置页保存的默认 Agent，并为当前 Town 和 Agent 自动维护一个稳定的浏览器会话，所以你可以边浏览边持续对话。

输入框会自动带上当前标签页引用。这个页面引用会跟随 Chrome 当前 tab 自动变化，也可以在输入框里删除。

在网页正文中选中文本时，选区旁会出现 `引用` 浮层。点击后，选中文本会作为输入框内的引用胶囊插入 Side Panel。

## 发送时实际发生了什么

发送消息时，扩展会按下面的顺序工作：

1. 读取当前网页标题和 URL
2. 抓取页面正文并转换成 Markdown
3. 生成一个附件文件
4. 发送给选定 Agent

## 常用快捷键

- `Cmd/Ctrl + Enter`：发送当前 Ask
- `Cmd + I`：打开 Downcity Side Panel；如果当前页面有选中文本，会作为引用插入输入框
- `Esc`：输入框聚焦时关闭 Downcity Side Panel
