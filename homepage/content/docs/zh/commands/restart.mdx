# `city restart`

重启完整的 city 运行栈。

当前行为顺序：

1. 停止 Console
2. 停止 city runtime
3. 重启 city runtime
4. 恢复重启前处于运行状态的托管 agent
5. 重新拉起 Console

## 用法

```bash
city restart
```

## 相关命令

- [`city start`](/zh/docs/commands/start)
- [`city stop`](/zh/docs/commands/stop)
- [`city status`](/zh/docs/commands/status)
- [`city console`](/zh/docs/commands/console/console)
