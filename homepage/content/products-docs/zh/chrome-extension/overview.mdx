# Chrome Extension 总览

Chrome Extension 是 Downcity Town 的浏览器入口。

它的主链路是右侧 Side Panel：你可以和 Agent 持续对话，并把当前页面或选中文本作为上下文引用。

## 适合什么场景

- 你正在读一篇文章，想在浏览器右侧直接和 Agent 讨论
- 你希望把某个网页快照发给 Agent 继续工作
- 你需要一个连接本机或远程 Downcity Town 的浏览器工具

## 它现在提供什么

### 1. Side Panel

Side Panel 负责：

- 打开一个常驻的浏览器对话
- 通过 `/api/sdk/*` 连接 Town runtime API
- 发送普通消息，并自动把当前页面作为输入框引用
- 在网页里选中文本后，通过选区旁的 `引用` 浮层插入选中文本
- 订阅 Agent 事件并展示回复

### 2. Settings Page

设置页负责：

- 设置 `Town URL`
- 选择默认 Agent
- 只在目标 Town 需要鉴权时填写 Town Token

## 它不会做什么

- 不再提供 Inline Composer
- 不再在网页内注入输入框
- 只保留用于选中文本引用的轻量页面浮层
- 设置页不再暴露多字段 Server 配置
