# `city start`

Start city runtime in background mode.

Startup initializes the city runtime registry and process state.
If no local token exists yet, startup automatically creates a `console-bootstrap` token and writes it into the local CLI auth state for this machine.

If you also want to start Console in one step, use `city start --console` or `city start -a`.

## Usage

```bash
city start
```

```bash
city start --console
```
