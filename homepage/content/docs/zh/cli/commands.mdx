# CLI 命令

## downcity (city)

### 全局初始化与更新

```bash
# 初始化 City 全局状态
downcity init

# 更新 CLI 到最新版本
downcity update
```

### Agent 生命周期

```bash
# 创建新 Agent 项目
downcity agent create my-project

# 列出已登记的 Agent 项目
downcity agent list

# 在后台启动 Agent
downcity agent start

# 在前台启动 Agent
downcity agent start --foreground

# 停止 Agent 守护进程
downcity agent stop

# 重启 Agent 守护进程
downcity agent restart

# 检查 Agent 状态
downcity agent status

# 诊断守护进程问题
downcity agent doctor
# 修复僵尸守护进程状态
downcity agent doctor --fix

# 与 Agent 对话
downcity agent chat
# 发送一次性消息
downcity agent chat -m "hello"
```

### Agent 历史

```bash
# 清理会话历史
downcity agent history clean
# 清理特定会话
downcity agent history clean --session-id <id>
```

### 环境变量

```bash
# 列出环境变量
downcity env list
# 设置环境变量
downcity env set KEY=value
# 删除环境变量
downcity env delete KEY
# 导出环境变量到 .env
downcity env copy
```

### Token

```bash
# 列出 API token
downcity token list
# 创建新 token
downcity token create
# 删除 token
downcity token delete <id>
```

### Plugin

```bash
# 交互式管理插件
downcity plugins
# 调度插件操作
downcity plugins schedule
```

### Chat

```bash
# 交互式管理 chat 资源
downcity chat
# 设置 chat 授权
downcity chat auth set <platform>:<userId>
```

## downfed

```bash
# 打开交互式 Federation 管理器
downfed

# 打开 admin 界面
downfed manage

# 添加 Federation 服务器
downfed server add

# 进入服务器工作区
downfed server manage

# 查询 Federation API
downfed query GET /health

# 创建新项目
downfed create my-project

# 部署项目
downfed deploy
# 带选项部署
downfed deploy --verify --dry-run
```
