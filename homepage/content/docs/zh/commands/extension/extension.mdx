# `city extension`

管理 extension 运行状态，并转发 extension command。

## 命令层级

- 一级命令：`city extension`
- 动作命令：`city voice`

`city voice` 属于 extension 体系动作命令，不是一级命令。

## 用法

```bash
city extension <subcommand>
```

## 子命令

- `list`
- `status <extensionName>`
- `start <extensionName>`
- `stop <extensionName>`
- `restart <extensionName>`
- `command <extensionName> <command> --payload <json>`

## 示例

```bash
city extension list
city extension status voice
city extension restart voice
city extension command voice status --payload '{}'
```

## 说明

需要先启动 agent runtime：

```bash
city agent start
```
