# `sma console`

`console` is the global control plane.

Use it to manage the console daemon and inspect managed agent runtimes.

## Usage

```bash
sma console start
sma console stop
sma console restart
sma console status
sma console agents [--json]
```

## Behavior

- `start`: start console daemon in background.
- `stop`: stop console daemon and managed child agent daemons.
- `restart`: restart console daemon.
- `status`: print console process state and active managed agents.
- `agents`: print active managed agents list.

## Typical Flow

```bash
# 1) start global console
sma console start

# 2) start project runtime
cd /path/to/agent-project
sma agent create .
sma agent on

# 3) check status
sma console status
sma agent status .
```
