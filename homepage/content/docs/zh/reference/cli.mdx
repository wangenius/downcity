## 运行模型（简版）

- `city start`：一键启动（依次执行 `city console start` 与 `city console ui start`）
- `city stop`：一键停止（等价 `city console stop`）
- `city restart`：一键重启（依次执行 `city console restart` 与 `city console ui start`）
- `city console start`：启动全局控制面
- `city agent create`：初始化项目运行文件
- `city agent start`：启动项目 agent daemon
- `city service ...`：运维核心 service 域与工作流
- `city plugin ...`：运维 plugin 的 action 与状态
- `city voice ...`：运维内置 voice plugin
- `city console model ...`：管理运行时模型配置
