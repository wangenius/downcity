# `city console start`

后台启动 console。

启动时会自动初始化空的 `~/.downcity/console/agents.json`，因此即使此时还没有任何 agent，Console UI 也会显示为正常的“空列表”状态，而不是配置缺失。

## 用法

```bash
city console start
```

## 相关命令

- [`city console stop`](/zh/docs/commands/console/stop)
- [`city console status`](/zh/docs/commands/console/status)
- [`city console agents`](/zh/docs/commands/console/agents)
