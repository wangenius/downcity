accounts service 不只给前端提供注册登录入口，也会暴露管理侧 service。

## 常见管理读取

```ts
const users = await admin.service("accounts").get("users");
const sessions = await admin.service("accounts").get("sessions");
```

## 这说明什么

说明 service 最终不是“前端专用功能”，而是：

- 一组用户态入口
- 一组管理侧读取或维护入口
- 一套底层用户 / session 数据事实

## 为什么这和 `Admin Gate` 有关系

因为这类动作不该在浏览器里做。

它们更适合：

- 后台管理系统
- 内部运营工具
- 安全的后端脚本
