# `city agent stop`

停止指定 agent 的后台 daemon。

## 用法

```bash
city agent stop [path]
```

## 行为说明

- `path` 默认是当前目录 `.`
- 目标 agent 需要已登记在 city registry
- 停止匹配的 daemon 进程组
- 清理 daemon pid/meta 文件
- 在 city registry 中把 agent 标记为 `stopped`
- 如果 daemon 没有运行，命令会成功退出并提示 not running

## 相关命令

- [`city agent start`](/zh/docs/commands/agent/start)
- [`city agent restart`](/zh/docs/commands/agent/restart)

## 示例输出

```text
$ city agent stop .
downcity v1.0.437

Agent daemon stopped                            lucas_whitman
  project   /Users/you/Documents/bots/lucas_whitman
  pid       12345
```
