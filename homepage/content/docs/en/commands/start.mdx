# `city start`

Start city runtime in background mode.

Startup initializes the city runtime registry and process state.
If unified auth has no users yet, startup prompts you to set the first admin password. Leaving it empty falls back to `downcity`, and the username is fixed as `admin`.

If you also want to start Console in one step, use `city start --console` or `city start -a`.

## Usage

```bash
city start
```

```bash
city start --console
```
