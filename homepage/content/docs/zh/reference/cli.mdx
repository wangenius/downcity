## 运行模型（简版）

- `sma console start`：启动全局控制面
- `sma agent create`：初始化项目运行文件
- `sma agent on`：启动项目 agent daemon
- `sma service ...`：运维核心 service 能力
- `sma extension ...`：运维 extension 扩展能力
- `sma config llm ...`：管理运行时模型配置
