## 通用问题

### ShipMyAgent 是什么？

ShipMyAgent 是一个 AI 驱动的开发助手，在您的代码库中本地运行。它帮助自动化代码审查、维护任务，并提供与仓库的自然语言交互。

### 它与 GitHub Copilot 有什么区别？

| 功能 | ShipMyAgent | GitHub Copilot |
|------|-------------|----------------|
| **重点** | 自主 agent | 代码补全 |
| **操作** | 运行定时任务 | 编辑器内建议 |
| **控制** | 完全配置 | 有限控制 |
| **交互** | 自然语言 | 代码补全 |
| **部署** | 自托管 | 云服务 |

### 我需要上传代码吗？

**不需要。** ShipMyAgent 完全在本地运行。您的代码永远不会离开您的机器，除了在发送到 LLM API 进行处理时。

### 支持哪些 LLM 提供商？

- ✅ Anthropic (Claude 3.5 Sonnet、Haiku、Opus)
- ✅ OpenAI (GPT-4o、GPT-4 Turbo)
- ✅ DeepSeek
- ✅ 任何兼容 OpenAI 的 API

### 系统要求是什么？

- **操作系统**：Windows、macOS、Linux
- **Node.js**：>= 18.0.0
- **内存**：最低 512MB，推荐 1GB
- **磁盘**：安装需要 100MB

### ShipMyAgent 是免费的吗？

是的！ShipMyAgent 是开源的，可以免费使用。您只需支付向所选提供商发出的 LLM API 调用费用。

### 我可以离线使用吗？

部分可以。Agent 需要互联网访问来与 LLM API 通信，但所有文件操作和任务调度都可以离线工作。

## 配置问题

### 如何更改 LLM 模型？

编辑 `ship.json`：

```json
{
  "llm": {
    "provider": "anthropic",
    "model": "claude-3-5-haiku-20241022"
  }
}
```

### 我可以使用多个 LLM 吗？

可以，配置备用模型：

```json
{
  "llm": {
    "primary": {
      "provider": "anthropic",
      "model": "claude-3-5-sonnet-20241022"
    },
    "fallback": [
      {
        "provider": "openai",
        "model": "gpt-4o"
      }
    ]
  }
}
```

### 如何设置环境变量？

**选项 1：Shell 配置文件** (~/.bashrc 或 ~/.zshrc)：
```bash
export ANTHROPIC_API_KEY="sk-ant-..."
export TELEGRAM_BOT_TOKEN="123456:..."
```

**选项 2：.env 文件**：
```bash
# 在项目根目录创建 .env
ANTHROPIC_API_KEY=sk-ant-...
TELEGRAM_BOT_TOKEN=123456:...
```

**选项 3：直接在 ship.json 中**（不推荐）：
```json
{
  "llm": {
    "apiKey": "sk-ant-..."  // ⚠️ 不要提交这个！
  }
}
```

## 集成问题

### 我必须使用 Telegram 吗？

不。Telegram 是可选的。您可以完全通过 CLI 使用 ShipMyAgent，无需任何集成。

### 如何设置 Telegram Bot？

1. 在 Telegram 上给 [@BotFather](https://t.me/BotFather) 发消息
2. 发送 `/newbot` 并按照说明操作
3. 复制 HTTP API token
4. 从 [@userinfobot](https://t.me/userinfobot) 获取您的 chat ID
5. 添加到 `ship.json`：

```json
{
  "services": {
    "telegram": {
      "enabled": true,
      "botToken": "${TELEGRAM_BOT_TOKEN}",
      "chatId": "12345678"
    }
  }
}
```

## 费用与成本

### 需要多少钱？

ShipMyAgent 是**免费的**。您只需支付 LLM API 使用费用。

**预估 LLM 成本**（每 100 万 token）：
- Claude 3.5 Haiku：~$0.25 输入 / $1.25 输出
- Claude 3.5 Sonnet：~$3 输入 / $15 输出
- GPT-4o：~$2.50 输入 / $10 输出

**典型月度使用**：
- 轻度使用：$5-20/月
- 中度使用：$20-50/月
- 重度使用：$50-200/月

### 如何降低成本？

1. **使用更快的模型**：Haiku 而不是 Sonnet
2. **优化提示**：保持 `PROFILE.md` 简洁
3. **限制任务频率**：不要过于频繁地运行任务
4. **使用缓存**：启用响应缓存
5. **设置 token 限制**：配置 `maxTokens`

### 我需要信用卡吗？

您需要向 LLM 提供商（Anthropic、OpenAI 等）提供付款方式。ShipMyAgent 本身不收取任何费用。

### 有免费套餐吗？

大多数 LLM 提供商提供免费套餐：
- Anthropic：新账户的免费额度
- OpenAI：提供免费套餐
- DeepSeek：非常低成本的替代方案

## 故障排查问题

### Agent 无法启动？

1. 检查 Node.js 版本：`node --version`（需要 >= 18）
2. 验证配置：`shipmyagent validate`
3. 检查日志：`shipmyagent logs --tail 50`
4. 参阅[故障排查指南](/docs/troubleshooting/common-issues)

### 命令不工作？

运行诊断：
```bash
shipmyagent doctor
```

### 遇到权限错误？


参阅[权限错误指南](/docs/troubleshooting/permission-errors)。

### LLM 无响应？

1. 测试 API key：`shipmyagent test-llm`
2. 检查 API 状态页面
3. 验证网络连接
4. 检查速率限制

### 任务未运行？

```bash
# 列出任务
shipmyagent tasks list

# 检查任务日志
shipmyagent logs --file tasks

# 手动运行
shipmyagent tasks run <task-id>
```
