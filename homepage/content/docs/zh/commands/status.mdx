# `city status`

查看 city runtime、Console 与当前托管 agent 概览。

## 用法

```bash
city status
```

## 相关命令

- [`city agent list --running`](/zh/docs/commands/agent/list)
- [`city console`](/zh/docs/commands/console/console)
