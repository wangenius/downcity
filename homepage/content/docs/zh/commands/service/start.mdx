# `city service start`

为指定 agent 目标启动某个 service。

日常使用更推荐直接走具体命令根，例如 `city memory start` 或 `city task start`。

## 用法

```bash
city service start <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
city service start task --agent my-agent
city memory start --path /abs/path/to/agent
```

## 相关命令

- [`city service stop`](/zh/docs/commands/service/stop)
- [`city service restart`](/zh/docs/commands/service/restart)

```text
$ city service start task
{
  "success": true,
  "service": "task"

```
