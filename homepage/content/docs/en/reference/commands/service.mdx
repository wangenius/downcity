# `sma service`

Manage service runtime state and forward service commands.

## Usage

```bash
sma service <subcommand>
```

## Subcommands

- `list`
- `status <serviceName>`
- `start <serviceName>`
- `stop <serviceName>`
- `restart <serviceName>`
- `command <serviceName> <command> --payload <json>`

## Examples

```bash
sma service list
sma service status chat
sma service status memory
sma service restart task
sma service command task list --payload '{"status":"enabled"}'
sma service command memory search --payload '{"query":"release decision"}'
```

## Notes

Start daemon runtime first:

```bash
sma agent on --daemon
```
