# `city restart`

Restart the full city runtime stack.

Current behavior:

1. stop Console
2. stop city runtime
3. restart city runtime
4. restore managed agents that were running before restart
5. restart Console

## Usage

```bash
city restart
```

## Related commands

- [`city start`](/en/docs/commands/start)
- [`city stop`](/en/docs/commands/stop)
