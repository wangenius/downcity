# Plugin State

你正在一个基于 plugin 的执行环境中工作。

## 可用命令总览
- plugin 状态管理（统一使用 `studio plugin`）：
  - `studio plugin list`
  - `studio plugin status <pluginName>`
  - `studio plugin start <pluginName>`
  - `studio plugin stop <pluginName>`
  - `studio plugin restart <pluginName>`
  - `studio plugin command <pluginName> <command> [--payload '<json>']`
  - `studio plugin schedule list`
  - `studio plugin schedule info <jobId>`
  - `studio plugin schedule cancel <jobId>`
- ActionSchedule 管理命令用于查看/取消延迟执行的 plugin action，不是独立 plugin。
- 当前内建托管 plugin：`shell` / `chat` / `task` / `memory` / `contact`。
- 当前内建本地 plugin：`auth` / `skill` / `asr` / `tts`。
- 模块级 action 命令：
  - `studio shell <action> ...`
  - `studio chat <action> ...`
  - `studio task <action> ...`
  - `studio memory <action> ...`
  - `studio plugin action <plugin> <action> ...`
  - `studio skill <action> ...`
  - `studio asr <action> ...`
  - `studio tts <action> ...`

具体 plugin 的执行参考对应文档。
