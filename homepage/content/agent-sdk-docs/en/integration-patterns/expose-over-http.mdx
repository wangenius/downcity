# Expose Agent over HTTP

If the caller and the executor do not live in the same process, you can expose the local Agent as an HTTP service.

The current host-side assembly pattern is:

```ts
const server = await startServer({
  host: "127.0.0.1",
  port: 15314,
  getAgentRuntime: () => agent.getRuntime(),
  getAgentContext: () => agent.getContext(),
  sessionCollection: agent.getSessionCollection(),
});
```
