# `city console stop`

停止 console（并停止其托管的子 agent runtime）。

## 用法

```bash
city console stop
```

## 相关命令

- [`city console start`](/zh/docs/commands/console/start)
- [`city console status`](/zh/docs/commands/console/status)

## 示例输出（基于当前实现）

以下示例直接对齐 `package/src` 当前命令实现（含默认输出模式与字段结构），示例值使用占位符表示。

```text
$ city console stop
city version: 1.0.0
✅ Console UI stopped
   pid: <uiPid>
   pidFile: <uiPidPath>
   log: <uiLogPath>
✅ DC console stopped
   pid: <pid>
   pidFile: <consolePidPath>
   log: <consoleLogPath>
```

```text
$ city console stop
city version: 1.0.0
ℹ️  DC console is not running
   pidFile: <consolePidPath>
   log: <consoleLogPath>
```
