# Built-in ChatPlugin Authorization

Chat authorization now belongs to `ChatPlugin`; it is no longer registered as a separate built-in plugin.

For concrete behavior, action names, and SDK usage, read [Agent Plugins Docs / chat](/en/plugins-docs/builtins/chat).

If you are looking for plugin concepts instead of one plugin manual, continue with:

- [Plugin Overview](/en/agent-sdk-docs/plugins/overview)
- [Plugin Design Patterns](/en/agent-sdk-docs/plugins/design-patterns)
- [Plugin Lifecycle](/en/agent-sdk-docs/plugins/lifecycle)
