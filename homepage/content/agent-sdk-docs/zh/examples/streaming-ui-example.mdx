# 例子：流式 UI

```ts
for await (const event of session.stream({
  query: "继续",
})) {
  if (event.type === "text-delta") {
    render(event.text);
  }
}
```

如果你还想显示工具过程，可以继续处理：

- `tool-call`
- `tool-result`
- `tool-error`

这个模式特别适合终端实时输出和网页流式聊天 UI。
