### 1. Cursor 还有必要换吗？

**没必要因为舆论换。范式没有改变，用的顺手就OK了。**不需要折腾迁移。本质上是一致的。都是 Agent。除非觉得它不顺手或者能力不行。

### 2. 播客剪辑 + 视频切片 + 多平台发布，这件事现在值不值得做？

这个问题没理解，是要做自己做内容播客，还是自己要做类似的工作流产品？

内容播客的话可以，工作流产品我个人觉得意义不大。直接用别人的产品就好。

### 3. 声音克隆现在效果还是不理想，能不能继续优化？

能优化，但短期很难做到“完全像真人口播”。这是大模型的底层能力，个人优化的意义不大。辛苦2-3个月做出来优化方案，然后大模型一迭代就cover了。而且现在可以把它从 60 分提到 75~85 分，但很难稳定到 95 分像真人。

### 6. 如果未来 AI 成为用户入口，独立站和 App 还有必要吗？贸易行业品牌主，要不要提前布局新平台入口？

**有的。 独立站是品牌名片，和生产力本身我觉得是不冲突的。 App 确实必要性会降低，很多App都会消失。**

### 7. 现在除了 GEO，还有没有实际案例/实践？

**有一个 cli(command line interface，计算机指令交互) for anything 的逻辑。 就是把你的产品中的一些action，封装成 agent 可以直接调用的命令接口。 比如： 查看你现在的产品有哪些？哪些在售，多少钱。甚至让 agent 可以直接买你的产品。**
