# Town Console

Town Console is the browser UI exposed by the Town gateway. Use it when you want to inspect and operate agents without staying in a terminal.

## Main Tasks

- start the console from [Town CLI](/en/products-docs/town-cli/console/console)
- inspect agent status
- manage plugin configuration and Town user City connection state
- check runtime state while debugging

For command details, read [`town console`](/en/products-docs/town-cli/console/console).
