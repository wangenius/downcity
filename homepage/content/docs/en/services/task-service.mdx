# Task Service

`task` service manages task definitions and execution.

## Responsibilities

- Create/update/delete task definitions
- Enable/disable/pause task status
- Run tasks immediately
- Manage cron runtime and scheduler reload

## Runtime operations

```bash
sma service status task
sma service command task list --payload '{"status":"enabled"}'
```
