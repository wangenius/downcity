# ShipMyAgent 文档

ShipMyAgent 把代码仓库启动为可对话、可执行、可审计的 Agent Runtime。

## 阅读路线

1. [快速开始](/zh/docs/quickstart/getting-started)
2. [配置说明](/zh/docs/quickstart/configuration)
3. [能力与集成](/zh/docs/capabilities)
4. [运维与排障](/zh/docs/operations)
5. [CLI / API 参考](/zh/docs/reference)

## 文档结构

- `quickstart`：安装、初始化、ship.json
- `capabilities`：Skills、Tasks
- `services`：Telegram、飞书、QQ 接入说明
- `extensions`：Voice 扩展使用说明
- `concepts`：架构、Service Runtime、Extension Runtime、关系与进程模型、消息链路
- `operations`：日志、故障排查、FAQ
- `reference`：命令参考、HTTP API、`.ship/` 目录

## 版本说明

本文档以当前仓库 `package` 中可执行 CLI 和类型定义为准，已清理旧版无效命令与过时页面。
