`AIService` is Downcity's built-in AI model service. One `AIService` manages all registered models and provides **two independent pathways** for each model.

## Two Pathways

```
                   Provider.model(...)
                         │
                    ai.use(model)
                         │
       ┌─────────────────┴─────────────────┐
       │                                   │
  SDK Pathway                   OpenAI-Compatible Pathway
  For User City                For downcity agent / curl / OpenAI SDK
       │                                   │
  POST /v1/ai/text      POST /v1/ai/chat/completions
  POST /v1/ai/stream         ↑
       │                     OpenAI-format body
       ▼                    { model, messages, stream, tools }
  provider.handlers.text                   │
  provider.handlers.stream                 ▼
    (ai-sdk wrapper)           provider.handlers.openai
       │                      (passthrough or format conversion)
       ▼                                │
    UIMessage                           ▼
  UIMessageStream                    Response
                                 (upstream raw response)
```

## Core Concepts

| Concept | Description |
|---|---|
| [Provider](/en/city-sdk-docs/service/ai/provider) | Encapsulates handler implementations and connection info for a third-party AI provider |
| [ModelConfig](/en/city-sdk-docs/service/ai/provider) | Model configuration (id, name, tags, handler bindings) generated by `provider.model()` |
| [AIService](/en/city-sdk-docs/service/ai/provider) | Model service instance. Registers models via `use()`, auto-registers routes |
| [Auto Passthrough](/en/city-sdk-docs/service/ai/chat-completions) | Zero-code support for OpenAI-compatible providers |
| [Format Conversion](/en/city-sdk-docs/service/ai/chat-completions) | Bidirectional adapters for non-OpenAI providers (Anthropic ↔ OpenAI) |
| [Calling](/en/city-sdk-docs/service/ai/calling) | Complete calling examples for both pathways |

## Quick Start

```ts
const ai = new AIService();
ai.use(deepseek.model({
  id: "deepseek-v4-flash",
  name: "DeepSeek V4 Flash",
  default: true,
}));
base.use(ai);

// SDK pathway
const result = await client.ai.text({ prompt: "hello" });

// OpenAI-compatible pathway (curl)
// POST /v1/ai/chat/completions
// { model: "deepseek-v4-flash", messages: [...], stream: true }
```
