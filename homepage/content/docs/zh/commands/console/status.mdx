# `city console status`

查看 console 进程状态、console UI 状态与当前托管 agent 概览。

## 用法

```bash
city console status
```

## 相关命令

- [`city console agents`](/zh/docs/commands/console/agents)
- [`city console`](/zh/docs/commands/console/console)
