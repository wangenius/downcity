# `sma service start`

启动目标 agent 下的指定 service。

## 用法

```bash
sma service start <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
sma service start task --agent my-agent
```

## 相关命令

- [`sma service stop`](/zh/docs/commands/service/stop)
- [`sma service restart`](/zh/docs/commands/service/restart)

以下示例直接对齐 `package/src` 当前命令实现（含默认输出模式与字段结构），示例值使用占位符表示。

```text
$ sma service start task
{
  "success": true,
  "service": "task"
}
```
