# Console CLI Commands Removed

Current `town` no longer provides Console UI startup commands. `town start` only starts local Town runtime; it does not start browser Console and does not accept Console-related options.

## What to use now

- Start local runtime: [`town start`](/en/products-docs/town-cli/start)
- Inspect runtime, City connection, and managed agents: [`town status`](/en/products-docs/town-cli/status)
- Manage agents: [`town agent`](/en/products-docs/town-cli/agent/agent)
- Use browser Console: start Console UI through its product deployment or development entry
