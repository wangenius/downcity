# Console CLI 命令已移除

当前版本的 `town` 不再提供 Console UI 启动命令。`town start` 只启动本机 Town runtime，不会启动浏览器 Console，也不支持 Console 相关参数。

## 当前应该怎么做

- 启动本机 runtime：[`town start`](/zh/products-docs/town-cli/start)
- 查看 runtime、City 连接和受管 agent：[`town status`](/zh/products-docs/town-cli/status)
- 管理 agent：[`town agent`](/zh/products-docs/town-cli/agent/agent)
- 使用浏览器 Console：按产品部署或开发入口单独启动 Console UI
