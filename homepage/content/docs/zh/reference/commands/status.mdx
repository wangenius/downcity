# `sma status`

查看 SMA manager 状态和活跃托管的 agent daemon。

## 用法

```bash
sma status
```

## 输出规则

- 显示 manager 进程是否在运行
- 显示 manager 的 `pidFile` 与 `log` 路径
- 显示当前活跃托管 agent daemon 概览

## 示例

```bash
# 查看 manager 全局状态
sma status

# 查看项目 daemon 状态（用 agent 子命令）
sma agent status /path/to/agent
```
