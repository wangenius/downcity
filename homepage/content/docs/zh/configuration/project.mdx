# 项目配置（ship.json）

`ship.json` 是项目级运行入口，常用字段如下：

```json
{
  "name": "my-agent",
  "model": {
    "primary": "default"
  },
  "start": {
    "host": "127.0.0.1",
    "port": 5314
  },
  "services": {},
  "permissions": {}
}
```

## 注意事项

1. `model.primary` 必须能在全局模型池中解析到。
2. `start.host/start.port` 会影响 CLI 的默认路由目标。
3. 项目层只维护 `services`；`extensions` 统一在 console 全局层配置。

## 常用命令

```bash
sma console config get
sma console config get model.primary
sma console config set start.port 3100
```
