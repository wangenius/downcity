---
title: AI Service
description: AIService 管理 AI 模型，提供两条独立通路：SDK 通路和 OpenAI 兼容通路。
---

`AIService` 是内建的 Service，通过 Provider 注册模型，为每个模型提供两条调用通路。

## 两条通路

| | SDK 通路 | OpenAI 兼容通路 |
|---|---|---|
| 路由 | `POST /v1/ai/text`, `/stream`, `/image`, ... | `POST /v1/ai/chat/completions` |
| 调用方 | `UserClient.ai.text()` | pi-agent, OpenAI SDK, curl |
| 输入格式 | `{ prompt, model }` | `{ messages, model, stream }` |
| 返回格式 | UIMessage / UIMessageStream | OpenAI 格式 JSON / SSE |
| Provider action | `text` / `stream` | `openai`（未提供则自动透传） |

## 快速开始

```ts
import { AIService, Provider } from "@downcity/infra";

// 创建 Provider
const deepseek = new Provider("deepseek", {
  baseURL: "https://api.deepseek.com/v1",
  envKey: "DEEPSEEK_API_KEY",
  text: myTextAction,
  stream: myStreamAction,
});

// 注册模型
const ai = new AIService();
ai.use(deepseek.model({ id: "deepseek-v4-flash", name: "DeepSeek V4 Flash" }));
base.use(ai);
```

注册后自动生成路由：

```
POST /v1/ai/text              → SDK 文本
POST /v1/ai/stream            → SDK 流式
POST /v1/ai/chat/completions  → OpenAI 兼容（自动透传）
```
