# Skill Service (Migrated)

`skill` no longer belongs to the service system. It has moved into the plugin system.

Use these pages instead:

- [`city skill`](/en/docs/commands/plugin/skill)
- [`city plugin`](/en/docs/commands/plugin/plugin)

## Migration summary

- `skill` has no independent lifecycle
- `skill` does not own a main workflow
- `skill` now exposes `find / install / list / lookup` through plugin actions
- the runtime skills overview system text is now provided by `skill plugin`
