# AgentRPC

`AgentRPC` 是本地 Agent 的 RPC 传输层，让其他进程或服务可以通过 Unix socket 或 TCP 端口访问 agent 的会话，无需走 HTTP。

需要额外安装 `@downcity/server`：

```bash
pnpm add @downcity/server
```

## 最小示例

```ts
import { AgentRPC } from "@downcity/server";
import { Agent } from "@downcity/agent";

const agent = new Agent({ id: "demo", path: process.cwd() });
const rpc = new AgentRPC(agent);

const binding = await rpc.listen();
```

## 构造参数

```ts
new AgentRPC(agent: Agent)
```

| 参数 | 类型 | 说明 |
|------|------|------|
| `agent` | `Agent` | 已构造好的本地 Agent 实例 |

## 方法

### `listen(options?)`

启动 RPC 监听。

```ts
const binding = await rpc.listen();
// 返回 AgentRpcBinding，包含连接端点信息
```

### `close()`

停止 RPC 服务并释放资源。

```ts
await rpc.close();
```
