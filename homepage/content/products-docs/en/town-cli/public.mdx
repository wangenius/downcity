# Public Console CLI Removed

Current `town` no longer provides Console public-exposure commands. `town start` only starts local runtime; it does not start or rebind Console UI.

If you need external Console access, expose Console UI through its product deployment, reverse proxy, tunnel, or cloud hosting layer.
