# Downcity Docs

Downcity 是给 AI builders 的 Agent 基础设施。这套文档帮助你运行本机 Agent、接入 City、连接模型和插件，并把 Agent 工作流做成真实产品表层。

最短产品模型：

- 产品负责用户体验
- Agent 在明确运行边界里执行任务
- Downcity 提供模型、工具、任务、记忆、服务、权限、usage、计费和控制台这些可复用运行层
