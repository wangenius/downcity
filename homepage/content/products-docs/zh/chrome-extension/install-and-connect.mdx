# 安装与连接

## 前置条件

在使用扩展前，先启动 Town 和 Agent：

```bash
town start
town agent start
```

如果你是从仓库本地加载扩展，还需要先构建产物：

```bash
npm run build:extension
```

## 在 Chrome 中加载

1. 打开 `chrome://extensions`
2. 打开右上角 `开发者模式`
3. 点击 `加载已解压的扩展程序`
4. 选择 `products/chrome-extension/dist`

## 第一次需要配置什么

打开扩展设置页，保持四步：

1. 填写 `Town URL`
2. 选择 `Agent`
3. 填写 `Default Ask`
4. 点击 `保存并检查`

本机开发时，Town URL 是：

```txt
http://127.0.0.1:5314
```

不要使用 `15314`。那是 Agent RPC 端口，不是浏览器使用的 HTTP 地址。

## Town Token

如果目标 Town 需要鉴权，设置页会显示 `Town Token`。

在本机终端创建 token：

```bash
town token create my-token
```

如果你更喜欢交互式流程，也可以执行：

```bash
town token
```

然后把得到的 token 粘贴到 `Town Token`，再点击 `保存并检查`。

## 远程 Town

如果连接远程服务，把远程可访问的 HTTPS 地址填到 `Town URL`。

确认这个地址可以被 Chrome 访问，并且指向 Town runtime HTTP API。

## 更新扩展后该做什么

每次改了 extension 代码后，先重新构建：

```bash
npm run build:extension
```

然后回到 `chrome://extensions`，点击扩展的刷新按钮。
