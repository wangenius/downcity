# Town Console

Town Console 是 Town gateway 暴露出来的浏览器界面。你可以不用一直停留在终端里，也能查看和操作 agents。

## 主要任务

- 通过 [Town CLI](/zh/products-docs/town-cli/console/console) 启动 console
- 查看 agent 状态
- 管理 plugin 配置，以及 Town user 的 City 连接状态
- 调试时检查 runtime 状态

命令细节见 [`town console`](/zh/products-docs/town-cli/console/console)。
