# `sma task`

任务系统命令组。

## 用法

```bash
sma task <subcommand> [options]
```

## 子命令

- `list`
- `create`
- `run <title>`
- `update <title>`
- `delete <title>`
- `status <title> <enabled|paused|disabled>`
- `enable <title>`
- `disable <title>`

## `create` 关键参数

- `--title <title>`（必填）
- `--description <description>`（必填）
- `--context-id <contextId>`（必填）
- `--when <when>`（默认 `@manual`，支持 cron / `time:ISO8601`）
- `--kind <agent|script>`
- `--status <enabled|paused|disabled>` 或 `--activate`
- `--body <body>`
- `--overwrite`

create 去重：只按 `title` 精确去重。

## `update` 常用参数

- `--title <title>` / `--description <description>`
- `--when <when>` / `--clear-when`
- `--kind <agent|script>`
- `--context-id <contextId>`
- `--status <status>` / `--activate`
- `--body <body>` / `--clear-body`

## `run` 返回语义

`run` 为异步受理，立即返回：

- `accepted=true`
- `message`
- `executionId`

## 示例

```bash
sma task list
sma task create --title "Daily check" --description "Run tests" --context-id telegram-chat-123 --when "0 10 * * 1-5" --activate
sma task run daily-check
sma task update daily-check --when "@manual"
sma task disable daily-check
```
