---
title: Usage 服务
description: 用 @downcity/services 自动记录 service 调用事件。
---

Usage 服务通过 service hook 记录调用事件。它只记录事实，不替产品决定价格、套餐或扣费规则。

## 启用

```ts
import { usageService } from "@downcity/services";

base.use(usageService({
  record_errors: true,
}));
```

## 查询事件

```ts
const events = await admin.service("usage").get("events");
const summary = await admin.service("usage").get("summary");
```

终端用户也可以查询自己的 usage：

```ts
const events = await user.service("usage").get("me");
```

`summary` 会按 bay、service 和状态汇总调用次数，适合管理后台快速展示总览。

服务会自动创建 `service_usage_events`，并记录 bay、user、service、model 和调用状态。
