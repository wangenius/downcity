# Chrome Extension 总览

Chrome Extension 是 Downcity Town 的浏览器入口。

它有两条主链路：右侧 Side Panel 和 Agent 持续对话，Popup 把当前网页整理成 Markdown 后发送给选定 Agent。

## 适合什么场景

- 你正在读一篇文章，想在浏览器右侧直接和 Agent 讨论
- 你希望把某个网页快照发给 Agent 继续工作
- 你需要一个连接本机或远程 Downcity Town 的浏览器工具

## 它现在提供什么

### 1. Side Panel

Side Panel 负责：

- 打开一个常驻的 Chrome Agent Session
- 通过 `/api/sdk/*` 连接 Town runtime API
- 发送普通消息，或附带当前页面 Markdown
- 订阅 Agent 事件并展示回复

### 2. Extension Popup

Popup 负责：

- 选择目标 Agent
- 输入 Ask 并发送
- 查看当前页面在当前 Town 下的最近发送历史

### 3. Settings Page

设置页负责：

- 设置 `Town URL`
- 选择默认 Agent
- 设置 Popup 默认 Ask
- 只在目标 Town 需要鉴权时填写 Town Token

## 它不会做什么

- 不再提供 Inline Composer
- 不再在网页内注入输入框
- Popup 不等待完整执行结果
- 设置页不再暴露多字段 Server 配置
