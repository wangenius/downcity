# `city service status`

查看某个 service 在静态 catalog 中的能力信息。

## 用法

```bash
city service status <serviceName> [--json]
```

## 示例

```bash
city service status chat
city service status memory --json
```

## 相关命令

- [`city service list`](/zh/docs/commands/service/list)
- [`city memory status`](/zh/docs/commands/service/memory)
