# 给本地 Agent 注入 ChatService

如果你希望 SDK 持有 chat service，可以直接传实例：

```ts
import { Agent, ChatService } from "@downcity/agent";

const agent = new Agent({
  id: "repo-helper",
  path: "/path/to/project",
  tools: {},
  services: [
    new ChatService({
      telegram: {
        botToken: process.env.TELEGRAM_BOT_TOKEN!,
      },
    }),
  ],
});
```

适合：

- 你在应用代码里自己掌握渠道生命周期
- 你不想依赖外部项目 runtime 自动装配

如果你希望 SDK 进程本身也具备渠道收发能力，这种显式注入方式会更可控。
