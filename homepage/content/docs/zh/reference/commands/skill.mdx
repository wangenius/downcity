# `sma skill`

管理 skills 的发现、安装与读取。

## 用法

```bash
sma skill <subcommand> [options]
```

## 子命令

- `find <query>`
- `install <spec>`
- `list`
- `lookup <name>`

## `install` 常用参数

- `-g, --global`：全局安装（默认 true）
- `-y, --yes`：跳过确认（默认 true）
- `--agent <agent>`：默认 `claude-code`

## 示例

```bash
sma skill find playwright
sma skill install owner/repo@playwright
sma skill list
sma skill lookup playwright
```

## 推荐流程

`find -> install -> lookup`
