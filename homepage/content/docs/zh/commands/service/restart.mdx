# `city service restart`

重启目标 agent 下的指定 service。

## 用法

```bash
city service restart <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
city service restart chat --agent my-agent
```

## 相关命令

- [`city service status`](/zh/docs/commands/service/status)
- [`city service command`](/zh/docs/commands/service/command)

## 示例输出（基于当前实现）

以下示例直接对齐 `package/src` 当前命令实现（含默认输出模式与字段结构），示例值使用占位符表示。

```text
$ city service restart task
{
  "success": true,
  "service": "task"
}
```

```text
$ city service restart unknown

  "success": false,
  "error": "<service error>"
}
```
