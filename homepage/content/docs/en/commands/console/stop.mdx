# `city console stop`

Stop console (and managed child agent runtimes).

The current implementation also tries to clean up detached orphan processes:

- `console run` processes that are still alive without a pid file
- `console ui run` processes that are still alive without a pid file
- agent daemons (`agent start --foreground true`) that are still alive without a pid file

## Usage

```bash
city console stop
```

## Related commands

- [`city console start`](/en/docs/commands/console/start)
- [`city console status`](/en/docs/commands/console/status)
