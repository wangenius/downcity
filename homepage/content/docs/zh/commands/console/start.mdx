# `city console start`

后台启动 console。

启动时会自动初始化空的 `~/.downcity/console/agents.json`，因此即使此时还没有任何 agent，Console UI 也会显示为正常的“空列表”状态，而不是配置缺失。
如果当前还没有任何统一账户用户，启动过程会提示设置首个管理员密码；留空时默认使用 `downcity`，用户名固定为 `admin`。

## 用法

```bash
city console start
```
