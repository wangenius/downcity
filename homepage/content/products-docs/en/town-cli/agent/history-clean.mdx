# `town agent history clean`

Hard-clean one targeted Agent session history.

This is a maintenance command for recovery and debugging. It is not a normal daily chat command, and it is not a vague "clear recent messages" shortcut.

## Usage

```bash
town agent history clean [path] --session-id <sessionId> --hard
town agent history clean [path] --channel telegram --chat-id 123 --hard
```

## Key options

- `--session-id`: target an exact session directly
- `--channel`: target by chat channel
- `--chat-id`: target by channel chat id
- `--target-type`: refine the target identity
- `--thread-id`: target a thread-specific history
- `--hard`: required for destructive cleanup
- `--json`: print structured output

## When to use it

Use this command only when you know exactly which session should be reset.

Typical cases:

- a session context is badly polluted
- one chat route must be reset
- you need a clean context for debugging

When you already know the exact session identity, `--session-id` is usually the cleanest targeting strategy.
