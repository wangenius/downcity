# CLI 命令

## Agent 生命周期

```bash
# 创建新项目
downcity agent init my-project

# 在当前目录启动 Agent
downcity agent start

# 停止运行中的 Agent
downcity agent stop

# 重启 Agent
downcity agent restart

# 检查状态
downcity agent status

# 查看日志
downcity agent logs
```

## City 身份

```bash
# 登录 City
downcity login

# 退出登录
downcity logout

# 查看当前身份
downcity whoami

# 列出 API token
downcity tokens list

# 创建新 token
downcity tokens create --name "ci-deployment"

# 撤销 token
downcity tokens revoke <token-id>
```

## City 用量与计费

```bash
# 查看当前用量
downcity usage

# 查看余额
downcity balance

# 充值（Stripe）
downcity balance add --amount 100
```

## 渠道账号

```bash
# 添加 Telegram 机器人
downcity chat add telegram --id telegram-main

# 添加飞书机器人
downcity chat add feishu --id feishu-work

# 列出账号
downcity chat list

# 移除账号
downcity chat remove telegram-main
```

## Federation 模型

```bash
# 列出可用模型
downcity models list

# 显示模型详情
downcity models show quality

# 列出提供商
downcity providers list

# 检查提供商健康
downcity providers status
```

## 全局标志

大多数命令支持：

- `--config <path>` — 使用自定义配置文件
- `--verbose` — 打印详细输出
- `--json` — 以 JSON 格式输出
