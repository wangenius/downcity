# `town status`

查看 Town runtime、City user 连接与当前托管 agent 概览。当前 Town CLI 不再展示 Console UI 运行态。

现在裸 `town` 会先进入全屏 TUI dashboard；这个命令仍然保留为可脚本化的文本状态入口。

## 用法

```bash
town status
```

## 相关命令

- [`town agent list --running`](/zh/products-docs/town-cli/agent/list)
- [`town city status`](/zh/products-docs/town-cli/city)

## 输出内容

- `Town runtime`：running、stale 或 stopped。
- `City connection`：当前 City base、town id 和 user token 状态。
- `Managed agents`：当前 registry 中仍在运行的 agent daemon。
