`@downcity/agent` 当前对外提供两个主要 SDK 入口：

- `Agent`：本地嵌入式 Agent
- `RemoteAgent`：连接已启动 Agent HTTP 服务的远程客户端

## 安装

```bash
pnpm add @downcity/agent
```

## 本地 Agent

当你的应用与 Agent 运行在同一进程，并且可以直接注入 tools、plugins 与 model 时，使用 `new Agent(...)`。

```ts
import { Agent } from "@downcity/agent";
import { createOpenAI } from "@ai-sdk/openai";

const openai = createOpenAI({
  apiKey: process.env.OPENAI_API_KEY!,
});

const agent = new Agent({
  id: "vibecape-official",
  path: "/path/to/agent-project",
  tools: {},
});

const session = await agent.createSession();
await session.set({
  model: openai.responses("gpt-5"),
});

const turn = await session.prompt({
  query: "把本地文件重命名为新的规范",
});
const result = await turn.finished;

console.log(result.text);
```

## 注入 ChatPlugin

如果你的 client 需要直接在 `@downcity/agent` 内启动 chat 渠道，可以显式传入 `ChatPlugin` 实例：

```ts
import { Agent, ChatPlugin } from "@downcity/agent";

const agent = new Agent({
  id: "vibecape-official",
  path: "/path/to/agent-project",
  tools: {},
  plugins: [
    new ChatPlugin({
      telegram: {
        botToken: process.env.TELEGRAM_BOT_TOKEN!,
      },
    }),
  ],
});
```

`plugins` 接收“已经实例化好的 plugin 对象”。  
`agent.start({ http: { ... }, rpc: true })` 会在暴露服务端点前自动启动这些 plugin。

## 观察执行过程

```ts
const unsubscribe = session.subscribe((event) => {
  if (event.type === "text-delta") {
    process.stdout.write(event.text);
  }
});

const turn = await session.prompt({
  query: "继续刚才的话题",
});
await turn.finished;
unsubscribe();
```

当前 session 事件类型包括：

- `text-delta`
- `reasoning-delta`
- `tool-call`
- `tool-result`
- `error`

## 分叉 Session

```ts
const branched = await session.fork();
const fromMessage = await session.fork("message-id");
```

传入 `messageId` 时，新 session 会复制到该消息为止（包含该消息）的历史。

## 启动 HTTP / RPC

```ts
const started = await agent.start({
  http: {
    host: "127.0.0.1",
    port: 15314,
  },
  rpc: true,
});

console.log(started.http?.baseUrl);
```

## 远程 Agent

当另一个进程已经调用过 `agent.start({ http: { ... } })` 时，使用 `RemoteAgent`。

```ts
import { RemoteAgent } from "@downcity/agent";

const agent = new RemoteAgent({
  baseUrl: "http://127.0.0.1:15314",
});

const session = await agent.createSession();

const turn = await session.prompt({
  query: "总结一下当前仓库结构",
});
const result = await turn.finished;

console.log(result.text);
```

远程观察方式相同：

```ts
const unsubscribe = session.subscribe((event) => {
  if (event.type === "text-delta") {
    process.stdout.write(event.text);
  }
});

const turn = await session.prompt({ query: "继续" });
await turn.finished;
unsubscribe();
```

## Session 落盘位置

SDK session 默认落盘到：

```text
<projectRoot>/.downcity/agents/<agentId>/sessions/<sessionId>/messages/messages.jsonl
```

这样即使多个 agent 共用同一个项目目录，session 也会按 `agentId` 隔离。
