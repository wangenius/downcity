# `town city`

`town city` 管理本机 Agent 调用 City 时使用的连接上下文。

Town 不配置 City 的模型、服务、账号、计费或管理端资源。这些仍然通过 `city` CLI 管理。

## 用法

```bash
town city
town city status
town city list
town city use
town city connect <url> --town-id <townId> --user-token <token>
town city disconnect
```

## 写入内容

`town city connect` 和 `town city use` 会写入 Town 平台 env：

- `DOWNCITY_CITY_URL`
- `DOWNCITY_CITY_TOWN_ID`
- `DOWNCITY_CITY_USER_TOKEN`

Agent runtime 需要调用 City 用户态服务时会读取这些值。

## 推荐流程

```bash
city
town city use
town city status
town agent start
```

用 `city` 添加 server、登录并管理共享资源。用 `town city use` 把当前 City 会话导入到 Town。
