# 创建第一个 Agent

如果你只想快速把一个项目启动成 agent，推荐直接按这条路径走：

```bash
bay agent create .
bay agent start
bay agent chat
```

如果这 3 步都顺利跑通，你再进入完整 Agent 文档会轻松很多，因为你已经具备了最基本的运行直觉。

- [创建第一个 Agent](/zh/docs/agent/getting-started/create-your-first-agent)
