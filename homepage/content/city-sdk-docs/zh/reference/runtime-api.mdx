## Federation 负责什么

- 初始化 Runtime 数据库
- 校验 `user_token`
- 提供统一的 `/v1/*` HTTP 路由
- 通过 `base.use(service)` 注册 Service、AIService 和服务
- 把 Service action 和服务路由挂到同一个 `/v1/*` 空间
- 通过 hook 接入 usage、套餐、余额、扣费、日志和风控

具体方法请继续查看 [Federation](/zh/city-sdk-docs/reference/federation)。
