# `city service list`

列出 console 级已注册 service 的静态目录。

## 用法

```bash
city service list [--json]
```

## 输出字段

- `name`：service 标识
- `scope`：当前作用域，目前为 `agent`
- `supportsLifecycle`：是否支持 lifecycle 控制
- `supportsCommand`：是否暴露 action 或自定义 command
- `actions`：静态注册的 action 名称

## 相关命令

- [`city service status`](/zh/docs/commands/service/status)
- [`city memory`](/zh/docs/commands/service/memory)
