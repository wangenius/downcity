# Packages 包

City SDK 文档只保留 `@downcity/city` runtime package，以及管理 Federation / 本机 Agent 时需要的 CLI 入口。

## City SDK 能力面

- [@downcity/city](/zh/city-sdk-docs/packages/city)：City、Federation、service/action 基建、hooks、store、table 和 client 访问层。
- [CLI](/zh/city-sdk-docs/packages/cli)：`fed/downfed` 管 Federation 部署和 admin 运维，`city/downcity` 管本机 Agent 和用户态连接。

## 相关 Space

- [Federation Services Docs](/zh/services-sdk-docs)：accounts、balance、usage、Stripe payment 等官方 service 包。
- [Products Docs](/zh/products-docs)：Downcity CLI、Console、Chrome Extension 产品手册。
- [Agent SDK](/zh/agent-sdk-docs)：可编程 Agent、Session、Local/Remote Agent 和 Plugin API。
