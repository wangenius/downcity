# SnapshotHttpPlugin 场景

## 场景

你想让某个 plugin 对外提供一个只读快照接口：

- `GET /api/notes/snapshot`

这类事情就很适合 `plugin.http.server`。

## 当前边界

这个能力针对的是完整 runtime HTTP 装配链。

也就是说：

- `plugin.http.server` 不是 `agent.plugins.runAction(...)` 这一类显式调用
- 当前由 Town 暴露 HTTP，并为运行中的 Agent 自动挂载 plugin HTTP 路由
- 如果你需要这一层，就应该把它理解成更完整的 host/runtime 集成能力
