## New runtime logic (short)

- `sma agent create`: create/initialize an agent project
- `sma start`: start manager in background
- `sma stop`: stop manager background process
- `sma manager status`: show manager + active managed agents
- `sma manager agents list`: list active managed agent daemons
- `sma status`: top-level alias for manager global status
- `sma agent on`: default background mode
- `sma agent on --foreground`: foreground mode
- `sma agent status`: check current directory agent status
- `sma agent off`: stop current directory agent
