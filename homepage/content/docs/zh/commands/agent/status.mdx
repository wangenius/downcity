# `city agent status`

查看指定 agent 的 runtime 状态。

## 用法

```bash
city agent status [path]
```

## 行为说明

- `path` 默认当前目录 `.`。
- 目标 agent 需要已登记在 console registry。
- 输出包含运行状态、进程信息和关键路径。
- 当状态为 `stale` 时，会额外输出 `stale_reason`，说明判定为僵尸状态的具体原因。

## 相关命令

- [`city agent start`](/zh/docs/commands/agent/start)
