# Example: Streaming UI

```ts
for await (const event of session.stream({
  query: "Continue",
})) {
  if (event.type === "text-delta") {
    render(event.text);
  }
}
```

If you also want to show tool activity, handle:

- `tool-call`
- `tool-result`
- `tool-error`

This pattern is especially good for terminal live output and web-based streaming chat UI.
