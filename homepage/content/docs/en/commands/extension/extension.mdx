# `city extension`

Manage extension runtime state and forward extension commands.

## Command hierarchy

- Top-level command: `city extension`
- Action command: `city voice`

`city voice` is an extension action command, not a top-level command.

## Usage

```bash
city extension <subcommand>
```

## Subcommands

- `list`
- `status <extensionName>`
- `start <extensionName>`
- `stop <extensionName>`
- `restart <extensionName>`
- `command <extensionName> <command> --payload <json>`

## Examples

```bash
city extension list
city extension status voice
city extension restart voice
city extension command voice status --payload '{}'
```

## Notes

Start agent runtime first:

```bash
city agent start
```

- [`city voice`](/en/docs/commands/extension/voice)
