# `sma chat`

chat service 命令组。

## 用法

```bash
sma chat <subcommand>
```

## 子命令

- `send`：发送消息到目标 `chatKey`
- `react`：给目标消息贴表情（当前主要 Telegram）
- `context`：查看当前会话上下文快照
- `history`：读取 chat 历史消息（默认最近 30 条）

## 示例

```bash
sma chat send --text "done"
sma chat react --emoji "✅"
sma chat context
sma chat history --limit 50
```

## 说明

- 该命令组面向 daemon 运行态，先启动 runtime：`sma agent on --daemon`。
