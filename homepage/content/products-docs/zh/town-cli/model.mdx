# `town model`

`town model` 已不再属于 Town 命令面。

Town 是本机 Agent 宿主和 Agent Plugin 管理器。City 模型由 `city` CLI 和 City runtime 管理。

请使用：

```bash
city
town city login
town city status
```

如果 Agent 项目需要在 `downcity.json.execution.modelId` 中填写 City 模型 ID，请从 City 管理侧选择模型，并让 Town 聚焦于托管 Agent 与管理 Plugin。
