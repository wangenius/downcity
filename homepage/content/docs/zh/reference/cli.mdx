## 运行模型（简版）

- `city start`：仅启动 city runtime
- `city start --console` / `city start -a`：同时启动 city runtime 和 Console
- `city console start`：只启动 Console，前提是 city runtime 已经运行
- `city stop`：先停止 Console，再停止 city runtime 与托管 agent
- `city restart`：重启 city runtime，恢复之前运行中的 agent，再重启 Console
- `city agent create`：初始化项目运行文件
- `city agent start`：启动项目 agent daemon
- `city console`：管理 Console 模块本身
- `city service ...`：运维核心 service 域与工作流
- `city plugin ...`：运维 plugin 的 action 与状态
- `city web ...`：运维内置 web plugin
- `city asr ...`：运维内置 asr plugin
- `city tts ...`：运维内置 tts plugin
- `city model ...`：管理运行时模型配置
