# `city env`

`city env` 用来管理 Console Env 条目。

## 用法

```bash
city env list [--scope global|agent|all] [--agent <agentId>] [--json]
city env set <key> <value> [--scope global|agent] [--agent <agentId>] [--description <text>] [--json]
city env delete <key> [--scope global|agent] [--agent <agentId>] [--json]
```

## 行为说明

- `list` 用来查看当前已经配置了哪些 key。
- `set` 用来新增或更新一个 key。
- `delete` 用来删除一个 key。
- 这些 key 的 value 存在 Console store 中，并按加密方式保存。
- `list` 不会返回任何明文 value。
- 传 `--agent <agentId>` 时会自动切换到 `agent` 作用域。
- `list --scope all` 会同时查看 global 和 agent 两层 env。

## 示例输出

```text
$ city env list --scope all
Env                                                2 configured
  OPENAI_API_KEY
    scope         global
    description   Shared OpenAI key
  TELEGRAM_BOT_TOKEN
    scope         agent
    agent         /Users/you/Documents/bots/lucas_whitman
```
