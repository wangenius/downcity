# CLI

Downcity 提供两个 CLI 二进制文件：

- **`downcity`**（别名 `city`）— 管理本地 Agent、全局配置和 City 状态。
- **`downfed`** — 管理 Federation 部署、City 实体和服务资源。

两者共享相同的全局标志（`--verbose`、`--quiet`、`--lang`）和帮助系统。

## 本地存储

- `downcity` 将本地 City 状态、加密凭证、环境变量、渠道账号和已登记 Agent 项目路径保存到 `~/.downcity/downcity.db`。
- `downfed` 将 Federation admin profile、admin key、Cloudflare account 选择和 CLI 偏好保存到 `~/.downcity/federation.db`。
- Agent 运行状态从每个项目的 `<project>/.downcity/.debug/` daemon meta/pid 文件推导。

## downcity

`downcity` 是本地管理 CLI。它处理 Agent 生命周期、插件、环境变量、token 和 Federation 连接。

### 全局命令

| 命令 | 说明 |
|---------|-------------|
| `downcity init` | 初始化 City 全局状态（`~/.downcity/downcity.db`） |
| `downcity update` | 更新 downcity CLI 到最新版本 |
| `downcity env` | 管理平台环境变量 |
| `downcity config` | 管理 City 配置 |
| `downcity token` | 管理 API token |
| `downcity federation` | 管理 Federation 连接 |

### Agent 命令

| 命令 | 说明 |
|---------|-------------|
| `downcity agent create [path]` | 创建并初始化 Agent 项目 |
| `downcity agent list` | 列出已登记的 Agent 项目 |
| `downcity agent start [path]` | 启动 Agent（后台/前台） |
| `downcity agent stop [path]` | 停止后台 Agent 守护进程 |
| `downcity agent restart [path]` | 重启后台 Agent 守护进程 |
| `downcity agent status [path]` | 查看后台 Agent 状态 |
| `downcity agent doctor [path]` | 诊断守护进程状态文件 |
| `downcity agent chat` | 在终端中与 Agent 对话 |
| `downcity agent history` | 管理 Agent 会话历史 |

### Plugin 命令

| 命令 | 说明 |
|---------|-------------|
| `downcity plugins` | 管理插件（交互式或列出操作） |
| `downcity plugins schedule` | 管理插件调度 |

### Chat 命令

| 命令 | 说明 |
|---------|-------------|
| `downcity chat` | 管理 chat 插件资源（交互式） |
| `downcity chat access` | 管理指定 Agent 的 Chat 用户准入 |

## downfed

`downfed` 是 Federation 管理 CLI。它处理部署、项目脚手架和管理操作。

| 命令 | 说明 |
|---------|-------------|
| `downfed` | 打开交互式 Federation 管理界面 |
| `downfed manage` | 打开 admin 管理界面 |
| `downfed server manage` | 进入 Federation 的 admin 工作区 |
| `downfed server add` | 添加已部署的 Federation URL |
| `downfed query <method> <path>` | 查询当前 active Federation 的 HTTP API |
| `downfed create [dir]` | 创建新 Federation 项目骨架 |
| `downfed deploy [source]` | 部署 Federation 项目 |

## 全局标志

- `--verbose` — 打印详细进度输出
- `--quiet` — 仅输出错误信息
- `--lang <locale>` — 设置 CLI 语言（zh/en）
- `--help` — 显示任何命令的帮助
