title: 模型与通路
description: Provider + AIService 两条独立通路的架构与工作原理。

Downcity 的 AI 服务有 **两条独立通路**，共用同一个模型注册：

```
                    Provider.model({ id: "deepseek-v4-flash", ... })
                                    │
                              ai.use(model)
                                    │
              ┌─────────────────────┴─────────────────────┐
             │                                           │
       SDK 通路                                    OpenAI 兼容通路
 给 User City 用                              给 downcity agent / curl / OpenAI SDK 用
             │                                           │
  POST /v1/ai/text                      POST /v1/ai/chat/completions
  POST /v1/ai/stream                         ↑
             │                            OpenAI 格式 body
             ▼                           { model, messages, stream }
  Provider text action                              │
  Provider stream action                            ▼
     (ai-sdk 封装)                       Provider openai action
             │                            (透传 或 格式转换)
             ▼                                    │
        UIMessage                               ▼
     UIMessageStream                         Response
                                          (上游原始响应)
```

## 自动透传

当 Provider 不提供 `openai` action 但提供了 `baseURL` + `envKey` 时：

```
POST /v1/ai/chat/completions
  body: { model: "deepseek-v4-flash", messages: [...], stream: true }
              │
              ▼
       resolve → model.actions.openai 不存在 → baseURL+envKey 存在
       → 自动透传
              │
              ▼
       fetch("https://api.deepseek.com/v1/chat/completions", {
         body: JSON.stringify(originalBody),   // 原样转发
       })
              │
              ▼
       上游 Response 原样返回（SSE 或 JSON）
```

## 格式转换

当 OpenAI 兼容上游需要自定义请求或响应处理时，可以写 `openai` action：

```ts
import { Provider, type Context, type AIProviderBilledResponse } from "@downcity/city";

class OpenAICustomProvider extends Provider {
  constructor() {
    super({
      id: "openai-custom",
      baseURL: "https://api.openai.com/v1",
      envKey: "OPENAI_API_KEY",
    });
  }

  async openai(ctx: Context): Promise<AIProviderBilledResponse> {
    const body = customizeOpenAIBody(ctx.input);
    const apiKey = ctx.env("OPENAI_API_KEY");
    const response = await fetch(`${this.baseURL}/chat/completions`, {
      method: "POST",
      headers: {
        authorization: `Bearer ${apiKey}`,
        "content-type": "application/json",
      },
      body: JSON.stringify(body),
    });
    return ctx.input.stream ? { response: convertStream(response) } : { response: convertJSON(await response.json()) };
  }
}

const openaiCustom = new OpenAICustomProvider();
```

## Provider 计费

Provider action 可以在正常输出旁边返回一条 billing。这样 provider 自己解析上游 usage 和价格，AIService / BillingService 不需要理解 token、cache、reasoning 等 provider 私有字段。

```ts
import { Provider, buildAssistantMessage, type Context, type AIProviderBilledOutput } from "@downcity/city";
import type { UIMessage } from "ai";

const ai = new AIService({
  billing: new BillingService({ balance }),
});

class PricedProvider extends Provider {
  constructor() {
    super({ id: "priced-provider" });
  }

  async text(ctx: Context): Promise<AIProviderBilledOutput<UIMessage>> {
    const result = await callUpstream(ctx.input);

    return buildAssistantMessage(result.text, ctx, {
      finishReason: "stop",
      usage: result.usage,
    }, {
      amount_microcredits: priceUpstreamUsage(result.usage),
      note: "priced-provider text",
      metadata: {
        provider_id: "priced-provider",
        raw_usage: result.usage,
      },
    });
  }
}

const provider = new PricedProvider();
```

流式调用可以返回一个 Promise，在上游 stream 结束后再 resolve 最终扣费：

```ts
return {
  response: stream.toUIMessageStreamResponse(),
  billing: stream.totalUsage.then((usage) => ({
    amount_microcredits: priceUpstreamUsage(usage),
    note: "priced-provider stream",
  })),
};
```

## SDK 调用

```ts
const client = new City({
  role: "user",
  city_url,
  town_id,
  user_token,
});
const result = await client.ai.text({ prompt: "hello", model: "deepseek-v4-flash" });
```
