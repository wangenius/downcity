# Example: Minimal RemoteAgent

```ts
import { RemoteAgent } from "@downcity/agent";

const agent = new RemoteAgent({
  baseUrl: "http://127.0.0.1:15314",
});

const session = await agent.createSession();
const turn = await session.prompt({
  query: "Summarize this project",
});

const result = await turn.finished;
```

It is best to use this minimal example to validate remote connectivity first, and only then add more complex subscription or multi-session behavior.
