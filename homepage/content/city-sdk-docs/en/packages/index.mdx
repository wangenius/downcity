# Packages

City SDK docs focus on the `@downcity/city` runtime package and the operator tooling around deployed City runtimes.

## City SDK Surface

- [@downcity/city](/en/city-sdk-docs/packages/city): City, CityBase, service/action infrastructure, hooks, store, table, and the client access layer.
- [City CLI](/en/city-sdk-docs/packages/cli): operator commands for a deployed City runtime.

## Related Spaces

- [City Services Docs](/en/services-sdk-docs): official service packages such as accounts, balance, usage, and Stripe payment.
- [Products Docs](/en/products-docs): Town CLI, Town Console, and Chrome Extension product manuals.
- [Agent SDK](/en/agent-sdk-docs): programmable agents, sessions, local/remote agent, and plugin APIs.
