# `sma config`

Manage project `ship.json` and structured LLM configuration commands.

## Usage

```bash
sma config <subcommand>
```

## Subcommands

- `get [keyPath]`
- `set <keyPath> <value>`
- `unset <keyPath>`
- `alias`
- `llm`

## `llm` subcommands

```bash
sma config llm provider list
sma config llm provider add <providerId>
sma config llm provider update <providerId>
sma config llm provider remove <providerId>

sma config llm model list
sma config llm model add <modelId>
sma config llm model update <modelId>
sma config llm model activate <modelId>
sma config llm model remove <modelId>
```

## Examples

```bash
sma config get llm.activeModel
sma config set start.port 3001
sma config unset services.chat.channels.telegram
sma config alias --dry-run
```
