# Agent CLI 总览

`city agent` 是单项目 agent 的命令入口。

## 你可以用它做什么

- 初始化一个 agent 项目
- 启动或停止后台 agent
- 查看已登记 agent 列表
- 在终端里和 agent 对话
- 清理会话历史
- 诊断和修复 stale daemon 状态

如果你把 Agent 看成“单项目运行时”，那 `city agent` 这组命令就是它的操作台。

## 推荐按功能理解命令

### 生命周期

- `create`
- `start`
- `stop`
- `restart`
- `status`
- `list`

这组命令回答的是：

- 项目有没有初始化
- 现在有没有跑起来
- 如何安全地启动、停止、重启

### 交互

- `chat`
- `history clean`
- 交互式 `city agent` manager

这组命令回答的是：

- 我怎么和 agent 说话
- 我怎么清理某条坏会话
- 我怎么通过交互式界面管理多个 agent

### 维护

- `doctor`
- daemon 状态排查
- registry 排查

这组命令回答的是：

- 为什么状态不对
- 为什么看起来已经启动过，但现在连不上
- 为什么 `list` 和 `status` 看起来不一致

## 一条推荐使用顺序

第一次接触 Agent CLI，建议按下面的顺序理解：

1. `create`
2. `start`
3. `chat`
4. `status`
5. `stop / restart`
6. `doctor`
