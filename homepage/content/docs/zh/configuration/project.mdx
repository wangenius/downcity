# 项目配置（downcity.json）

`downcity.json` 是项目级运行入口，常用字段如下：

```json
{
  "name": "my-agent",
  "model": {
    "primary": "default"
  },
  "start": {
    "host": "127.0.0.1",
    "port": 5314
  },
  "services": {}
}
```

## 注意事项

1. `model.primary` 必须能在全局模型池中解析到。
2. `start.host/start.port` 会影响 CLI 的默认路由目标。
3. 项目层统一在 `downcity.json` 中维护 `services`、`plugins`、`assets`。

## 常用命令

```bash
city console config get
city console config get model.primary
city console config set start.port 3100
```
