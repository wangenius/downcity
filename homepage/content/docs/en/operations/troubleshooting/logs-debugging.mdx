# Logs & Debugging

## 1) Follow runtime logs

```bash
tail -f .downcity/logs/$(date +%F).jsonl
```

## 2) Inspect service state

```bash
city service list
city plugin list
```

## 3) Inspect conversation state

```bash
city chat context --chat-key ctx_xxxxxxxxxxxxxxxx
city chat history --chat-key ctx_xxxxxxxxxxxxxxxx --limit 50
```

## 4) Inspect task artifacts

```bash
ls -la .downcity/task/<taskId>
ls -la .downcity/task/<taskId>/<timestamp>
```

`run.json` and `result.md` usually reveal the root cause quickly.
