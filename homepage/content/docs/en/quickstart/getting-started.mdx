# Getting Started

## 1. Install

```bash
npm install -g downcity
downcity --version
```

> `downcity` and `city` are the same CLI binary.

## 2. Initialize Console (first time only)

```bash
city console init
```

This creates console-global config under `~/.ship/`.

## 3. Start Console

```bash
city console start
```

> `city agent start` requires console to be running first.

## 4. Initialize a repository

```bash
cd /path/to/your-repo
city agent create .
```

Initialization generates:

- `PROFILE.md` (required)
- `SOUL.md` / `USER.md` (optional)
- `ship.json`
- `.ship/` runtime directory

## 5. Configure LLM

Create `.env` at project root (do not commit):

```bash
LLM_API_KEY=your_key
```

Minimal `ship.json` example:

```json
{
  "name": "my-agent",
  "version": "1.0.0",
  "model": {
    "primary": "default"
  }
}
```

## 6. Start Agent runtime

Start runtime (default: background daemon):

```bash
city agent start
```

Use foreground mode when you want logs in the current terminal:

```bash
city agent start --foreground
```

## 7. Verify runtime state

```bash
city agent status
city console agents
```

Notes:

- Agent ports are auto-allocated by console, so Quickstart should not assume a fixed port.
- In multi-agent setups, each agent can have a different runtime port.

## 8. Optional: start Console UI

```bash
city console ui start
```

## 9. Optional: Chrome extension quick-send from page selection

If you use the repo's `chrome-extension`:

1. Select text on the page, then click the message button near the selection to open the inline input near the selected area.
2. Clicking the browser extension action icon opens the extension popup; press `Cmd/Ctrl + U` to open the inline input on the page.
3. In popup mode, use the `Ask History` selector to quickly refill recent asks.
4. No extra quote `tag` is shown; the current selection is used directly as send context.
5. Type `/` to quickly pick from recent ask history.
6. Press `Cmd/Ctrl + Enter` or click the send button to submit (`Enter` for newline).
7. Press `Esc` to close the input box.
8. If nothing is selected, sending automatically falls back to full-page mode.
