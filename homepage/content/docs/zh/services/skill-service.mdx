# Skill Service

`skill` service 负责本地已学习技能的管理。

## 职责

- 查找 skill
- 安装 skill
- 列出已学会 skill
- 查阅 `SKILL.md` 并注入执行上下文

## 运行态操作

```bash
sma service status skill
sma service command skill list --payload '{}'
```
