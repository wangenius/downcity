## Runtime model (short)

- `sma console start`: start global control plane
- `sma agent create`: initialize project runtime files
- `sma agent start`: start project agent daemon
- `sma service ...`: operate service domain capabilities
- `sma extension ...`: operate extension capabilities
- `sma console model ...`: manage model config for runtime
