# 第一次和 Agent 对话

启动 agent 后，直接运行：

```bash
town agent chat
```

如果当前环境里有多个运行中的 agent，CLI 会让你选择目标；如果你想稳定复现，建议显式写 `--to <id>`。

如果你想深入了解 CLI chat、渠道接入和 SDK 用法，继续看：

- [CLI chat](/zh/docs/agent/cli/interaction/chat)
- [接入聊天渠道](/zh/docs/agent/getting-started/connect-chat-channels)
- [Agent SDK](/zh/agent-sdk-docs)
