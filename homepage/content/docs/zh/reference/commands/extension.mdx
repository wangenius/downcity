# `sma extension`

extension runtime 管理命令组。

## 用法

```bash
sma extension <subcommand>
```

## 子命令

- `list`：列出全部 extension 运行状态
- `status <extensionName>`：查看单个 extension 状态
- `start <extensionName>`：启动 extension
- `stop <extensionName>`：停止 extension
- `restart <extensionName>`：重启 extension
- `command <extensionName> <command>`：转发 extension command 载荷

## 示例

```bash
sma extension list
sma extension status voice
sma extension restart voice
sma extension command voice status
```

## 说明

- 该命令组面向 daemon 运行态，先启动 runtime：`sma agent on --daemon`。
