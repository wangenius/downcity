# `city console`

`console` 是全局控制面。

你可以通过它管理 console 运行状态，并查看当前托管的 agent runtime。

## 命令层级

一级命令组为：

- `city console`
- `city agent`
- `city service`
- `city plugin`

`console` 下面再细分 `init/model/config/start/stop/status/agents/ui`。

## 用法

```bash
city console init
city console model <subcommand>
city console config <subcommand>
city console start
city console stop
city console restart
city console status
city console agents [--json]
city console ui [start|stop|restart|status] [--port 5315]
```

## 行为说明

- `init`：初始化全局 console 配置（`~/.downcity/*`）。
- `model`：独立管理 console 全局语言模型池（provider/model/test/pause）。
- `config`：管理项目级 `downcity.json` 与 alias。
- `start`：后台启动 console。
- `stop`：停止 console，并停止其托管的子 agent runtime。
- `restart`：重启 console，并自动恢复重启前正在运行的托管 agent。
- `status`：查看 console 进程状态与活跃 agent 清单。
- `agents`：打印当前活跃托管 agent 列表。
- `ui`：管理单实例 UI（start/stop/restart/status），并可在页面内切换查看不同 agent。

## 典型流程

```bash
# 1) 启动全局 console
city console start

# 2) 启动项目 runtime
cd /path/to/agent-project
city agent create .
city agent start

# 3) 查看状态
city console status
city agent status .
```
