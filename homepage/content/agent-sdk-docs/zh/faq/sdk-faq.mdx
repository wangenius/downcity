# SDK FAQ

## 为什么本地 session 需要先 `set({ model })`？

因为本地 SDK Agent 负责的是执行壳和会话能力，具体模型实例由调用方显式传入。

## 为什么 RemoteAgent 不能远程传本地模型实例？

因为模型实例通常不可序列化，应由服务端本地持有。

## 为什么本地 SDK 和项目 runtime 的模型配置方式不一样？

因为它们面向的使用场景不同：

- 项目 runtime 面向长期运行的项目执行体，模型来自 `execution.modelId`
- 本地 SDK 面向嵌入式调用，模型实例由调用方显式注入
