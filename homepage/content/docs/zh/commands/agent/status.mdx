# `bay agent status`

查看指定 agent 的进程状态。

> 这页保留为命令入口。更完整的说明见 [CLI / status](/zh/docs/agent/cli/lifecycle/status) 和 [Agent 显示 stale](/zh/docs/agent/operations/troubleshooting/agent-shows-stale)。

## 用法

```bash
bay agent status [path]
```

## 行为说明

- `path` 默认当前目录 `.`。
- 目标 agent 需要已登记在 console registry。
- 输出包含运行状态、进程信息和关键路径。
- 当状态为 `stale` 时，会额外输出 `stale_reason`，说明判定为僵尸状态的具体原因。

## 相关命令

- [`bay agent start`](/zh/docs/commands/agent/start)
- [`bay agent restart`](/zh/docs/commands/agent/restart)
- [`bay agent list --running`](/zh/docs/commands/agent/list)
