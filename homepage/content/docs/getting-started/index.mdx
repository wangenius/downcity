## Framework Architecture

downcity is built around four core components that work together to create intelligent AI agents:

### ğŸ¤– Hero
The AI agent interface that handles conversations and decision-making. Heroes can be configured with different models, personalities, and capabilities.

### ğŸ  Room  
Persistent memory storage that maintains context across sessions. Rooms contain multiple conversation threads and long-term memory.

### ğŸ’¬ Shot
Individual conversation sessions within a Room. Shots manage message history and session-specific context.

### ğŸ“– Codex
Vector-based knowledge storage for semantic search and retrieval. Codex enables agents to access external knowledge and documents.
