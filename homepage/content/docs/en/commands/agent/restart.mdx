# `sma agent restart`

Restart runtime for a target agent.

## Usage

```bash
sma agent restart [path] [options]
```

## Key options

- `-p, --port <port>`: service port
- `-h, --host <host>`: service host

## Behavior

- `path` defaults to current directory `.`
- Target agent must be registered in console registry
- Restart flow is stop first, then start with latest options
