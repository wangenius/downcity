# Chrome Extension Overview

The Chrome extension is Downcity's web dispatch entry point.

Its job is simple: capture the current page as a Markdown attachment, then send it to a chosen Downcity Server, Agent, and Session.

## Best Use Cases

- You are reading an article and want an Agent to summarize, translate, or extract actions
- You want to send a live page snapshot into an existing Session
- You have multiple local or remote Servers and want to switch between them quickly

## What It Provides

### 1. Extension Popup

The popup lets you:

- choose the active Server Connection
- choose the target Agent
- choose the target Session
- write an Ask and send it
- review recent page history for the current connection

### 2. Settings Page

The settings page lets you:

- create multiple Server Connections
- configure Host, Port, and Bearer Token for each connection
- choose default Agent / Session per connection
- set a default Ask for the popup

## What It No Longer Does

- no Inline Composer
- no on-page input panel
- no waiting for the final execution result inside the popup
- no replacement for the original Session UI
