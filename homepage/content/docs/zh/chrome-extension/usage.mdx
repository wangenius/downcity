# 使用说明

## Extension Popup 怎么用

1. 点击浏览器工具栏中的扩展图标
2. 选择目标 Agent
3. 选择目标 Channel Chat
4. 输入任务说明
5. 点击发送，或按 `Cmd/Ctrl + Enter`

Extension Popup 适合做明确、完整的一次投递。

## 页内输入框怎么用

### 方式 1：先选区再点按钮

1. 在网页中选中一段文字
2. 选区附近会出现消息按钮
3. 点击后打开页内输入框

### 方式 2：快捷键打开

按 `Cmd/Ctrl + U` 可以直接打开页内输入框。

如果当前没有选区，扩展会自动退化为全页模式。

Inline Composer 会读取当前选区或页面正文，并支持两条路径：

- `Agent 投递`：沿用原来的 Agent + Channel Chat 投递链路
- `模型直答`：调用设置页中配置的默认模型，直接在页面内显示回复

输入框底部保留原来的 `Agent / Channel` 选择按钮，旁边新增一个纯图标模式按钮，用来在这两种模式之间切换。扩展会记住你上一次的选择。

## 常用快捷键

- `Cmd/Ctrl + U`：打开页内输入框
- `Cmd/Ctrl + Enter`：发送
- `Esc`：关闭页内输入框

如果 slash 建议面板已打开，`Esc` 会先关闭它。

## 发送时实际发生了什么

### 选区模式

- 读取当前选中的文本
- 包装成 Markdown 附件
- 连同你的任务说明一起投递给目标 context

### 全页模式

- 从多个正文候选区块中挑选最像正文的内容
- 合并多个强相关主体区块
- 解析图片引用
- 生成 Markdown 快照并上传

## 页面历史有什么用

Extension Popup 会记录当前页面最近的发送历史，便于你：

- 回看之前发过什么
- 快速复用类似问题
- 减少重复输入
