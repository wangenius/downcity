# `sma skill`

管理 skills 的发现、安装与读取。

## 用法

```bash
sma skill <subcommand> [options]
```

## 子命令

- `find <query>`
- `install <spec>`
- `list`
- `lookup <name>`

## `install` 常用参数

- `-g, --global`：全局安装（默认 true）
- `-y, --yes`：跳过确认（默认 true）
- `--agent <agent>`：默认 `claude-code`

## 示例

```bash
sma skill list
sma skill lookup playwright

# 仅当 list 中缺失时再安装
sma skill find playwright
sma skill install owner/repo@playwright
sma skill lookup playwright
```

## 推荐流程

`list -> lookup`（缺失时再 `find -> install -> lookup`）

## 运行时行为说明

- 在 Agent 运行时，`lookup` 的工具返回只包含执行状态。
- 技能正文会通过下一条 user message 注入，并使用 `<skill>...</skill>` 包裹。
