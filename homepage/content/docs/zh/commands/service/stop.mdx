# `city service stop`

停止目标 agent 下的指定 service。

## 用法

```bash
city service stop <serviceName> [--agent <name> | --path <path>] [--host <host>] [--port <port>]
```

## 示例

```bash
city service stop task --agent my-agent
```

## 相关命令

- [`city service start`](/zh/docs/commands/service/start)
- [`city service status`](/zh/docs/commands/service/status)
