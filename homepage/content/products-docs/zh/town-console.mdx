# Town Console

Town Console 是 Downcity 的浏览器控制台界面。当前 `town` CLI 不再负责启动 Console UI；`town start` 只启动本机 runtime。

## 主要任务

- 查看 agent 状态
- 管理 plugin 配置，以及 Town user 的 City 连接状态
- 调试时检查 runtime 状态

如果你在仓库内开发 Console UI，可以使用 workspace 的 Console 开发入口；部署环境则按产品部署方式提供浏览器访问。
