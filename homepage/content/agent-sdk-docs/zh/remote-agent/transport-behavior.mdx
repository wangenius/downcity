# 传输行为

RemoteAgent 当前通过 HTTP 调用服务端的 SDK 路由。

## 普通请求

例如：

- 创建 session
- 列出 session
- `run`
- `history`
- `fork`

这些都走普通 JSON 请求响应。

## 流式请求

`stream()` 走的是 NDJSON 风格的流式返回。

也就是说，客户端会不断收到一行一行的 JSON 事件。

每一行都可以看成一条独立的 stream event。

## 对调用方意味着什么

- 如果你只是写脚本，用 `run()` 更简单
- 如果你要做实时 UI，优先用 `stream()`

如果你要展示工具调用过程、渐进输出或流式聊天 UI，`stream()` 通常明显比 `run()` 更合适。
