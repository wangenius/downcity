# `city voice`

`city voice` 是内置 voice plugin 的短命令组。

## 它负责什么

- 启用或关闭 voice plugin
- 管理 voice 的转写配置
- 切换当前转写模型
- 转写本地音频文件

## 常用示例

```bash
# 启用 voice，并安装模型
city voice on SenseVoiceSmall

# 查看 plugin + transcriber 状态
city voice status

# 切换当前模型
city voice use whisper-large-v3-turbo

# 转写一个本地音频文件
city voice transcribe ./demo.ogg
```

## 说明

- `city voice on` 会在一步里完成“启用 plugin + 可选安装转写依赖”。
- voice plugin 会接入 `chat` service 定义的语音相关扩展点，Telegram / QQ 的语音消息会通过这些点完成转写与正文增强。
- 配置会持久化到项目 `downcity.json` 的 `plugins.voice`。
