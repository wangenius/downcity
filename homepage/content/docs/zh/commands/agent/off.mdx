# `sma agent off`

停止指定 agent 的 runtime。

## 用法

```bash
sma agent off [path]
```

## 行为说明

- `path` 默认当前目录 `.`。
- 目标 agent 需要已登记在 console registry。
- 找不到登记项时会直接报错并退出。
