# `sma console config`

管理项目 `ship.json`，并提供 console 全局 `llm` 结构化管理命令。

> 命令名说明：配置命令是 `sma console config`，不是 `sma config`。

## 用法

```bash
sma console config <subcommand>
```

## 子命令

- `get [keyPath]`
- `set <keyPath> <value>`
- `unset <keyPath>`
- `alias`
- `llm`

## `llm` 子命令

> `llm` 子命令读写的是 `~/.ship/ship.db`（console 全局 SQLite），不是项目 `ship.json`。

```bash
sma console config llm provider list
sma console config llm provider add <providerId>
sma console config llm provider update <providerId>
sma console config llm provider remove <providerId>

sma console config llm model list
sma console config llm model add <modelId>
sma console config llm model update <modelId>
sma console config llm model remove <modelId>
```

## `extensions.*` 路径说明

当 `keyPath` 以 `extensions.` 开头时，读写目标自动切换为 console 全局 `~/.ship/ship.json`。

```bash
sma console config get extensions.voice
sma console config set extensions.voice.enabled true
```

## 示例

```bash
sma console config get model.primary
sma console config set model.primary fast
sma console config set start.port 3001
sma console config unset services.chat.channels.telegram
sma console config llm model update quality --temperature 0.5
sma console config alias --dry-run
```
