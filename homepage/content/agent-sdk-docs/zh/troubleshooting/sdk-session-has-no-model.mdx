# SDK Session 没有模型

这是本地 SDK 最常见的问题之一。

## 原因

本地 SDK session 默认没有模型，除非宿主主动提供。

你可以用任一方式修复：

```ts
const agent = new Agent({
  id: "repo-helper",
  path: "/path/to/project",
  model,
});
```

或者：

```ts
await session.set({ model });
```

## 正确顺序

```ts
const session = await agent.session();
const turn = await session.prompt({ query: "继续" });
await turn.finished;
```

如果你已经确认本地 `Agent` 创建成功，但执行仍然失败，这通常就是第一优先排查点。

## 远程 session 呢

远程 session 当前不支持远程传本地模型实例。
