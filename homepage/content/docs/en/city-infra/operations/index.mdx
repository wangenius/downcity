Downcity matters not only because it makes the first call work, but because it lets that call stay stable while serving multiple products.

The operations section focuses on reliability, key rotation, usage observability, and troubleshooting.

## When to read this section

- you already finished Quickstart and are about to serve real users
- multiple products are starting to share one InfraRuntime
- you want to locate issues precisely between `token / model / provider / hook / env`
