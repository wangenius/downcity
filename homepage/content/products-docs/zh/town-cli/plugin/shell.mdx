# shell built-in

Shell 执行现在是 Agent 内建能力，不再是 plugin command surface。运行中的 Town agent 会通过 Agent HTTP gateway 给 SDK 客户端暴露 shell 审批 API。

## 推荐路径

使用 Agent SDK：

```ts
const approvals = await agent.approvals();
await agent.approve({ approval_id: approvals[0].approval_id });
await agent.deny({ approval_id: approvals[0].approval_id });
```

远程客户端：

```ts
import { RemoteAgent } from "@downcity/agent";

const agent = new RemoteAgent({ url: "http://127.0.0.1:5314" });
const approvals = await agent.approvals();
```

## 说明

- `@downcity/plugins` 不再提供 `ShellPlugin`。
- Shell tools 通过 `new Agent({ shell: new Shell() })` 挂载。
- Unrestricted sandbox 请求会以 `tool-approval-request` session event 出现。
- 每次审批只覆盖一个命令或一次 `shell_write`。
