# 内建 Services

当前 Downcity runtime 注册表里有 5 个内建 service：

- `chat`
- `contact`
- `task`
- `memory`
- `shell`

这页现在只作为入口。具体的逐项说明，已经拆到独立的 `builtins/` 子模块里，结构和 built-in plugins 一致。

## 先分清两层概念

### runtime 内建 service

这是 Downcity 完整 runtime 已经注册好的那一层 service。

### SDK 显式装配 service

这是 `@downcity/agent` 使用者可以主动装配进本地 `Agent` 的那一层能力。

这两层有重叠，但不是完全等价。

当前最直接的 SDK 显式入口仍然主要是：

- `ChatService`
- `BaseService`（用于 custom service）

这次新增的 built-ins 文档，会把整套 runtime 内建 service 都用 SDK 视角重新组织清楚，包括那些更偏 runtime 形态、但同样很重要的 service。
