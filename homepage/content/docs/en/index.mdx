# Downcity Docs

Downcity is agent infrastructure for AI builders. These docs help you run local agents, connect City, connect models and plugins, and turn agent workflows into real product surfaces.

The shortest product model:

- products own the user experience
- agents execute work inside explicit runtime boundaries
- Downcity provides the reusable runtime layer for models, tools, tasks, memory, services, permissions, usage, billing, and control surfaces
