# `city start`

后台启动 city runtime。

启动时会初始化 city runtime 的 registry 和进程状态。
如果当前还没有任何本机 token，启动过程会自动初始化一个 `console-bootstrap` token，并把它写入当前机器的 CLI 鉴权状态。

如果你希望一并启动 Console，请使用 `city start --console` 或 `city start -a`。

## 用法

```bash
city start
```

```bash
city start --console
```
