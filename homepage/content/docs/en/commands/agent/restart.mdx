# `studio agent restart`

Restart the process for a target agent.

> This page remains as a command entry point. For the fuller explanation, see [CLI / restart](/en/docs/agent/cli/lifecycle/restart) and [Restart an Agent Safely](/en/docs/agent/operations/runbook/restart-safely).

## Usage

```bash
studio agent restart [path] [options]
```

## Key options

- `-h, --host <host>`: daemon host

## Behavior

- `path` defaults to current directory `.`
- Target agent must be registered in console registry
- Restart flow is stop first, then start again with a new console-assigned port
