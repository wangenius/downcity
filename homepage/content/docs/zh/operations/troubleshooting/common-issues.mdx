# 常见问题

## 命令找不到（`downcity: command not found`）

```bash
npm config get prefix
export PATH=$(npm config get prefix)/bin:$PATH
```

或直接用：

```bash
npx downcity --version
```

## 端口冲突（5314 被占用）

```bash
lsof -nP -iTCP:5314 -sTCP:LISTEN
city agent doctor . --fix
city agent restart .
```

## 启动后无法调用 `city task/chat/...`

先确认 daemon 运行中：

```bash
city agent start
city service list
```

## LLM 配置错误

检查：

```bash
city console model list
city console config get model.primary
```

并确认：

1. `model.primary` 指向 `~/.downcity/downcity.db` 中存在的 model id
2. console 全局 SQLite 的 providers/models 记录完整可用
