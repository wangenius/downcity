## Suggested order

1. Building a new product first: read [Pure-client products](/en/city-sdk-docs/guides/pure-client-towns)
2. Already operating multiple products: continue with [Many products, one shared runtime](/en/city-sdk-docs/guides/multi-town-city)
3. Deploying on Cloudflare Workers: continue with [Cloudflare Workers](/en/city-sdk-docs/guides/cloudflare-workers)
4. Ready for real users: read [Auth and `user_token`](/en/city-sdk-docs/guides/auth-and-user-token)
5. Ready for real providers: continue with [Provider environment](/en/city-sdk-docs/guides/provider-env) and [Model routing](/en/city-sdk-docs/guides/model-adapters)
