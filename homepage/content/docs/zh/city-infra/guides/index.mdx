这些指南不是 API 参考，而是“在什么场景下应该怎么用 Downcity”。

如果你已经理解了基本概念，但开始接真实产品、真实用户和真实 provider，这一组文档会更有用。

## 推荐顺序

1. 如果你在做新产品，先看 [纯客户端产品](/docs/city-infra/guides/pure-client-products)
2. 如果你已经有多个产品，接着看 [多产品共用 InfraRuntime](/docs/city-infra/guides/multi-product-infra)
3. 如果你准备部署到 Cloudflare Workers，接着看 [Cloudflare Workers](/docs/city-infra/guides/cloudflare-workers)
4. 如果你要开始接真实用户，再看 [鉴权与 user_token](/docs/city-infra/guides/auth-and-user-token)
5. 如果你要开始接真实 provider，再看 [provider 环境变量](/docs/city-infra/guides/provider-env) 和 [模型路由](/docs/city-infra/guides/model-adapters)
