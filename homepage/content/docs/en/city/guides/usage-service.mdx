---
title: Usage Service
description: Use @downcity/services to record service call events automatically.
---

The usage service records service events through hooks. It records facts only; your bay still decides pricing, plans, and charging rules.

## Enable

```ts
import { usageService } from "@downcity/services";

base.use(usageService({
  record_errors: true,
}));
```

## Query Events

```ts
const events = await admin.service("usage").get("events");
const summary = await admin.service("usage").get("summary");
```

End users can query their own usage:

```ts
const events = await user.service("usage").get("me");
```

`summary` groups counts by bay, service, and status, which is useful for admin dashboards.

The service creates `service_usage_events` automatically and records bay, user, service, model, and call status.
