# `city keys`

`city keys` manages Console Env entries.

## Usage

```bash
city keys list [--scope global|agent|all] [--agent <agentId>] [--json]
city keys set <key> <value> [--scope global|agent] [--agent <agentId>] [--description <text>] [--json]
city keys delete <key> [--scope global|agent] [--agent <agentId>] [--json]
```

## Behavior

- `list` shows configured key names and metadata.
- `set` creates or updates a key.
- `delete` removes a key.
- Values are stored encrypted in the Console store.
- `list` never returns secret values.
- If `--agent <agentId>` is provided, scope becomes `agent` automatically.
