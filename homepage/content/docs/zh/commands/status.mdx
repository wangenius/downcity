# `studio status`

查看 Studio runtime、Console 与当前托管 agent 概览。

## 用法

```bash
studio status
```

## 相关命令

- [`studio agent list --running`](/zh/docs/commands/agent/list)
- [`studio console`](/zh/docs/commands/console/console)
- [`studio public`](/zh/docs/commands/public)
