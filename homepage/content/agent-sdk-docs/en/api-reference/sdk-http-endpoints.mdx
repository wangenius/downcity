# SDK HTTP Endpoints

These SDK routes do not belong to `Agent` itself.

They are exposed by `AgentHTTP` from [`@downcity/server`](/en/agent-sdk-docs/local-agent/http-and-rpc). Mount them as a standalone HTTP server, or as a Hono sub-router on any host server (Downcity gateway included).

The current core set includes:

- `GET /api/sdk/sessions`
- `POST /api/sdk/sessions`
- `GET /api/sdk/sessions/:sessionId`
- `POST /api/sdk/sessions/:sessionId/prompt`
- `GET /api/sdk/sessions/:sessionId/events`
- `GET /api/sdk/sessions/:sessionId/messages`
- `GET /api/sdk/sessions/:sessionId/system`
- `GET /api/sdk/sessions/:sessionId/approvals`
- `GET` / `POST /api/sdk/sessions/:sessionId/approval-mode`
- `POST /api/sdk/sessions/:sessionId/approval`
- `POST /api/sdk/sessions/:sessionId/fork`
- `POST /api/sdk/sessions/:sessionId/archive`
- `GET` / `DELETE /api/sdk/archived-sessions`
- `POST /api/plugins/action`
