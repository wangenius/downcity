# `city console init`

初始化全局 console 配置。

## 用法

```bash
city console init [--force]
```

## 行为

- 写入/更新 `~/.downcity/downcity.db`（全局配置 + 模型池）
- 生成 `~/.downcity/schema/downcity.schema.json`（编辑器 schema）

## 常用参数

- `--force`：覆盖已有全局配置（危险操作）

## 说明

- 这是 console 级命令，不依赖 agent 项目路径。
- 后续项目级配置请使用 [`city console config`](/zh/docs/commands/console/config)。
