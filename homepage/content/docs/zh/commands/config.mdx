# `city config`

管理项目 `downcity.json` 与 alias。

> 命令名说明：配置命令是 `city config`，不是 `city console config`。

## 用法

```bash
city config <subcommand>
```

## 子命令

- `get [keyPath]`
- `set <keyPath> <value>`
- `unset <keyPath>`
- `alias`

> 模型池管理已迁移到独立命令：[`city model`](/zh/docs/commands/model)。

## Plugin 配置

`city config` 现在只读写项目 `downcity.json`。

插件相关配置请改为：

- `plugins.*` 同时管理插件行为与插件自己的依赖配置

示例：

```bash
city config get plugins.asr
city config set plugins.asr.enabled true
city config set plugins.asr.modelId '\"SenseVoiceSmall\"'
```

## 示例

```bash
city config get model.primary
city config set model.primary fast
city config unset services.chat.channels.telegram
city config alias --dry-run
```
